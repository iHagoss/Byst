Benutzer abmelden

Diese Funktion wechselt in das Abmeldemenü. Zum Abmelden des Benutzers die Schaltfläche Abmelden wählen.

Vorgehen:

  1. Schaltfläche wählen.
  2. Schaltfläche wählen. Der Anmeldedialog wird angezeigt.
  3. Im Anmeldedialog die Schaltfläche Abmelden wählen. Der aktuelle Benutzer wird abgemeldet.
Sprachversion bei angemeldeten Benutzer ändern
  1. Schaltfläche wählen.
  2. Schaltfläche wählen.
  3. Schaltfläche wählen.
  4. Die Schaltfläche Sprache wählen wählen.
  5. Im Auswahldialog die gewünschte Sprache wählen. Das Programm wird automatisch auf die gewählte Sprache umgestellt.
