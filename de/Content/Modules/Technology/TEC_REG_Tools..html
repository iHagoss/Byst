Werkzeuge

In diesem Register werden die Biegewerkzeuge bestimmt, mit denen die Biegungen erstellt werden sollen. Die Zuweisung der Biegewerkzeuge kann automatisch oder manuell vorgenommen werden.

Der aktuell gewählte Biegeprozess Als Biegeprozess werden alle Handlungen oder Abläufe bezeichnet, die für das Biegen eines oder mehrerer Biegungen notwendig sind. wird in der 2D-Anzeige als grüne Biegelinie Als Biegelinie wird eine Linie bezeichnet an der entlang die Biegung der Schenkel erfolgt. angezeigt. Jedem Biegeprozess können ein oder mehrere Biegeoperationen zugeordnet sein.

Der Biegeprozess wird durch Klicken auf eine Biegelinie gewählt.

Werkzeuganzeige für Biegeprozesse

Unterhalb der 2D-Anzeige wird die Zuweisung der Werkzeuge zu den einzelnen Biegeprozessen tabellarisch angezeigt. Durch Klicken auf den Biegeprozess wird der Dialog Bearbeiten geöffnet.

Stehen für das verwendete Biegewerkzeug Warnungen oder Fehler an, wird dies durch ein Symbol hinter dem Biegewerkzeug angezeigt. Durch Bewegen der Maus über das Warnsymbol, werden Information zur Warnung angezeigt.

Tip: Kann die Warnung oder der Fehler für den Biegeprozess ignoriert werden, können diese im Dialog Bearbeiten quittiert werden (See «Schaltflächen»).

NOTE: Wird die Meldung nicht quittiert, muss ein anderes, passendes Werkzeug für den Biegeprozess gewählt werden.

Biegemethode Als Biegemethode werden das 3-Punkt Biegen, Luftbiegen und Prägen bezeichnet.

Zeigt die Biegemethode an, mit welcher der Prozess gebogen wird.

Biegeoperation

Zeigt die Art der Biegung Die Biegung ist ein Element der Profilprogrammierung und bezeichnet eine Formänderung im Werkstück. Einer Biegung können mehrere Attribute zugeordnet werden: Schenkellänge, Biegewinkel, Biegeradius (optional), Anbiegewinkel (optional), zusätzliche Attribute (optional). Eine Biegung kann aus mehreren Biegeprozessen bestehen. im Biegeprozess an.

  • Normalbiegung Die Normalbiegung ist die übliche Methode, eine Biegung zu erstellen. Die Biegung wird durch den Biegewinkel und einem optionalen Anbiegewinkel bestimmt.
  • Flachbiegung Dieser Biegetyp biegt das Blech mit einem Biegewinkel von 180° flach zusammen. Der Biegevorgang wird nacheinander in den zwei Schritten «Vorbiegen» und «Flachbiegen» vollzogen. Diese Schritte können auch getrennt voneinander ausgeführt werden.
  • Vorbiegung Dieser Biegetyp erstellt eine Biegung in zwei Schritten. Die Biegung wird zuerst mit einem Anbiegewinkel gebogen und zu einem späteren Zeitpunkt mit dem Endwinkel fertig gebogen. Verwenden Sie das Vorbiegen bei komplexen Biegeteilen, um Kollisionen während dem Biegeprozess zu vermeiden.
  • Hilfsbiegung Die Hilfsbiegung wird bei bestimmten Biegeformen vorläufig eingefügt und dann wieder zurückgebogen, um die Endform eines Biegeteils zu ermöglichen.
  • Stepradiusbiegung

Winkel

Zeigt den Winkel der Biegung an. Der Winkel 0 wird nur bei der Endbiegung einer Flachbiegung verwendet.

Oberwerkzeug

Unterwerkzeug

Zeigt das Unterwerkzeug das für den Prozess verwendet wird, sowie den Status des Werkzeugs mit einem Symbol an.
