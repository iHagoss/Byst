Zugewiesen

In diesem Dialog werden alle Biegeprozesse angezeigt, die einem Einrichtplan Der Einrichtplan besteht aus mindestens einem Stationsplan. Jeder Stationsplan besteht wiederum aus mindestens einer Biegestation auf dem das Biegeteil gebogen wird. Auf einem Einrichtplan sind alle Werkzeuge für den gesamten Biegeprozess definiert. und einer Station Eine Station besteht aus ein- oder mehreren Segmenten eines Ober- und Unterwerkzeugs. zugewiesen sind. Die Prozesse können der zur Verfügung stehenden Biegestationen und Einrichtplänen neu zugewiesen werden.

Erklärung zu den einzelnen Schaltflächen:

  • Konflikte quittieren

    • Es können Warnungen und Fehler für Prozess, Biegung Die Biegung ist ein Element der Profilprogrammierung und bezeichnet eine Formänderung im Werkstück. Einer Biegung können mehrere Attribute zugeordnet werden: Schenkellänge, Biegewinkel, Biegeradius (optional), Anbiegewinkel (optional), zusätzliche Attribute (optional). Eine Biegung kann aus mehreren Biegeprozessen bestehen., Station oder Teil quittiert werden.

  • Warnungen

    • Mit dieser Funktion wird die Warnung des gewählten Biegeprozesses quittiert. Die Warnung wird hinter der Werkzeugbezeichnung als Symbol angezeigt.

    Tip: Verwenden Sie diese Funktion für Warnungen, die für den gewählten Biegeprozess Als Biegeprozess werden alle Handlungen oder Abläufe bezeichnet, die für das Biegen eines oder mehrerer Biegungen notwendig sind. ignoriert werden können. Mit dem Quittieren wird dem System mitgeteilt, dass das Werkzeug für den Biegeprozess verwendet werden kann.

  • Fehler

    • Mit dieser Funktion können Fehler von zugewiesenen Werkzeugen quittiert werden.

    Info: Die Funktion ist nur aktiv, wenn der anstehende Fehler quittiert werden kann.

  • Zuweisung aufheben

    • Hebt die Zuweisung zum einzelnen Prozess, der aktuellen Station oder dem gesamten Teil auf. Die aufgehobenen Zuweisungen werden in das Register Nicht zugewiesen verschoben.

  • Versatz

    • Öffnet den Dialog in dem die Position des Biegeteils auf der Station definiert werden kann (See «Versatz»).

Biegeprozess einer anderen Station zuweisen
  1. Den Biegeprozess wählen, der neu zugewiesen werden soll.
  2. Die Schaltfläche Zuweisung aufheben wählen. Der Biegeprozess wird in das Register Nicht zugewiesen verschoben.
  3. Die neue Biegestation Eine Biegestation besteht immer aus einem oberen und unteren Biegewerkzeug und ist Bestandteil eines Stationsplans. im Dialog Einrichtplan wählen. Sie wird grün hervorgehoben.
  4. Im Register Nicht zugewiesen den Biegeprozess wählen, welcher der neuen Biegestation zugewiesen werden soll.
  5. Die Schaltfläche Zuweisen wählen. Der Biegeprozess wird der Biegestation neu zugewiesen und in das Register Zugewiesen verschoben.

Tip: Das Zuweisen kann auch durch Wählen des Biegeprozesses und gleichzeitiges Ziehen über die gewünschte Biegestation durchgeführt werden.

Erklärung zu den einzelnen Spalten:

Prozess

Zeigt die Reihenfolge der Biegeprozesse mit einer Laufnummer an.

Einrichtplan

Zeigt die Nummer des Einrichtplans an, auf dem der Prozess gebogen wird.

Station

Zeigt die Nummer der Biegestation an, auf welcher der Prozess gebogen wird.

Versatz

Zeig die Position des Biegeteils ausgehend von der linken Seite der Biegestation an.
