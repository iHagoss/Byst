Versatz

In diesem Dialog können Sie die Position des Teils auf der Biegestation Eine Biegestation besteht immer aus einem oberen und unteren Biegewerkzeug und ist Bestandteil eines Stationsplans. festlegen. Das Festlegen verschiedener Positionen dient zur Vermeidung von Kollisionen bei der Berechnung der Biegungen.

Attention! Wird die Position auf der Biegestation bei bereits gesetzten Anschlägen verändert, muss im Register Anschläge für den Prozess die Funktion Nachpositionieren ausgeführt werden (See «Nachpositionieren»).

Zum Positionieren eines Biegeteils auf der Station Eine Station besteht aus ein- oder mehreren Segmenten eines Ober- und Unterwerkzeugs. gibt es folgende Möglichkeiten:

  • Links

    • Verschiebt das Biegeteil an das linke Ende der Station. Die Option wird verwendet, damit Laschen beim Biegen nicht mit der Station kollidieren.

  • Zentriert

    • Zentriert das Biegeteil in der Station. Die Option wird standardmässig beim Luftbiegen Bei dieser Biegemethode wird das Biegteil mit dem Oberwerkzeug in das Unterwerkzeug gedrückt. Die Werkstückform wird im wesentlichen durch die relative Lage der Werkzeuge bestimmt. Durch unterschiedlich vorgegebene Eindringtiefen des Oberwerkzeugs in das Unterwerkzeug, werden verschiedene Biegewinkel erreicht. verwendet.

  • Rechts

    • Verschiebt das Biegeteil an das rechte Ende der Station. Die Option wird verwendet, damit Laschen beim Biegen nicht mit der Station kollidieren.

  • Benutzerdefiniert

    • Verschiebt das Biegeteil an eine frei wählbare Position. Die Option wird für folgende Situationen verwendet.

    • Bei Stationen mit Leersegmenten zum Anpassen der Werkstückposition an die Station
    • Beim 3-Punkt Biegen Das 3-Punkt Biegen bewirkt eine wiederholbare und sehr genaue Winkeleinstellung über das Keilsystem. Der Biegewinkel wird durch die beiden Einlaufkanten, des Unterwerkzeugs und dem Unterwerkzeugboden bestimmt. Die 3 Punkte liegen auf der gleichen Seite des Bleches. Die Winkeleinstellung erfolgt über höhenverstellbaren Bolzen im Unterwerkzeug. Die Verformung des Stössels wird durch das Hydrokissen ausgeglichen. Jedes der segmentierten Oberwerkzeuge arbeitet autonom gegen das Hydrokissen und wird gegen Überlast abgesichert. Der Biegevorgang ist erst beendet, wenn das Material auf dem Unterwerkzeugboden aufliegt und das Oberwerkzeugin das Hydrokissen eingetaucht ist., um die 75%-Regel Ist als Biegemethode «3-Punkt» gewählt, muss die Belastung eines Werkzeugsegments mindestens 75% betragen. Wird dies nicht beachtet, kann auf Dauer das Hydrokissen der 3-Punkt Biegemaschine schaden nehmen! Kann die Belastung durch Verwendung eines anderen Segments nicht auf mindestens 75% erhöht werden, muss die Biegemethode auf Luftbiegen eingestellt werden. einzuhalten.
  • Prozess

    • Weist die gewählte Position nur dem aktuellen Prozess zu.

  • Station

    • Weist die gewählte Position allen Prozessen auf der Station zu.
