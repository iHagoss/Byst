Biegeteil importieren

In diesem Register können Biegeteile in das Menü Home importiert werden. In der Eingabezeile kann durch Angabe des Teilenamens die Anzeige gefiltert werden.

Dateityp
  • Alle bekannten Dateiformate

    • Mit dieser Funktion kann ein Biegeteil, dass im Format .bpx oder .bendpart erstellt wurde, aus einer Datenablage importiert werden.

  • BPX-Dateien

    • Mit dieser Funktion kann ein Biegeteil, dass mit MMC ByBend oder BySoft 6.x erstellt wurde, aus einer Datenablage werden.

  • Bendpart-Dateien

    • Mit dieser Funktion kann ein Biegeteil, dass mit dem CAD/CAM-Programm BySoft 7 erstellt wurde, aus einer Datenablage importiert werden.

  • DXF-Dateien

    • Mit dieser Funktion kann ein Biegeteil im DXF-Format mit Biegeinformationen aus einer Datenablage importiert werden.

Sortieren nach
  • Name

    • Sortiert die Biegeteile nach dem Teilenamen.

  • Datum

    • Sortiert die Biegeteile nach dem Erstelldatum.

  • Grösse

    • Sortiert die Biegeteile nach der Dateigrösse.

  • aufsteigend

    • Beginnend vom Anfang des Alphabets.

  • aufsteigend

    • Beginnend vom Ende des Alphabets.

Importieren

Importiert die gewählten Biegeteile in den Teilepool des Programms. Es kann nur jeweils ein Teil importiert werden.

Konflikte beim Import

Treten beim automatischen Import von *.bendpart Dateien Namenskonflikte auf, werden diese angezeigt.

Es gibt folgende Lösungsmöglichkeiten:

  • Die Ursprungsdatei löschen. Die Datei wird nicht importiert.
  • Datei importieren und bestehendes, gleichnamiges Biegeteil überschreiben.

    • Das bereits geladene Teil wird automatisch entladen und gelöscht. Danach wird das neue Teil importiert und automatisch unter gleichem Namen geladen.

    Attention! Dabei gehen alle Änderungen der Technologie, Korrekturen, etc. vom geladenen Teil verloren.

Favoriten setzen

Speichert den ausgewählten Pfad als Favorit. Dieser Pfad wird in diesem Register immer als Standardpfad angezeigt.

Tip: Favoriten vereinfachen das Auffinden von zu importierenden Biegeteilen, die unter einem Standardpfad gespeichert sind.
