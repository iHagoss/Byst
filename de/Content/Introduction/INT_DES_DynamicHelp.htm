Online-Hilfe anzeigen

Die Online-Hilfe ist integrierter Bestandteil des Programms. Entsprechend der eingestellten Sprachversion des Betriebssystems, wird das Programm und die Online-Hilfe in der jeweiligen Sprachversion angezeigt.

Anzeigemöglichkeiten der Hilfe

In diesem Programm gibt es zwei verschiedene Arten, wie die Hilfe über das Hauptmenü aufgerufen werden kann.

Symbol Bedeutung
Gesamt-Hilfe

Zeigt die Hilfe in einem eigenen Hilfesystem an. Zur Navigation im Hilfesystem stehen folgende Möglichkeiten zur Verfügung:

  • Inhaltsverzeichnis
  • Indexverzeichnis
  • Glossar
  • Volltextsuche
Kontext-Hilfe

Zeigt die Hilfe in einem integrierten Hilfe-Dialog im Programm an. Der Inhalt wird zum gewählten Softwareelement kontextsensitiv angezeigt.

Durch wählen des Pins im Hilfedialog kann die Kontextfunktion ein- oder ausgeschaltet werden.

  • Kontext-Hilfe aktiviert
  • Kontext-Hilfe deaktiviert
Gesamt-Hilfe aufrufen
  1. Schaltfläche wählen.
  2. Schaltfläche wählen. Die Hilfe wird als eigenständige Anwendung geöffnet.
Kontext-Hilfe aufrufen
  1. Schaltfläche wählen.
  2. Schaltfläche wählen. Im eingeblendeten Dialog werden die Informationen zur gewählten Ansicht oder Funktionen angezeigt.
Hilfe zu Funktionen in Kopfzeile anzeigen
  1. Schaltfläche wählen.
  2. Schaltfläche wählen. Der Hilfedialog wird eingeblendet.
  3. Auf das gewünschte Menüsymbol in der Kopfzeile tippen. Das Hilfethema zur Funktion wird angezeigt.
  4. Den Finger ohne vom Bildschirm abzuheben auf einen Bereich im Hilfedialog ziehen, der nicht von dem Menü verdeckt wird (See «Ziehen»).
  5. Den Finger vom Bildschirm entfernen.
  6. In der Hilfe auf das Symbol der gewünschten Funktion tippen.
