Iniciar sesión en el programa

Tras iniciar el software aparece la pantalla principal con el diálogo Inicio de sesión. Es imprescindiblemente necesario seleccionar un usuario del carrusel de usuarios para poder iniciar sesión en el programa. Adicionalmente hay que introducir un código PIN de 4 dígitos.

Iniciar sesión de usuario

  1. Seleccionar usuario o indicar el código PIN.
  2. Seleccionar botón Iniciar sesión. El programa cambia a la vista Preparar Job.

Info: Al iniciar la sesión quedan disponibles las funciones correspondientes a los derechos del usuario.

NOTE: La descripción de las actividades y vistas que pueden ejecutarse con determinados derechos de usuario se describen en las instrucciones de funcionamiento de la máquina. Los derechos de usuario pueden adaptarse en el programa Administrador.

Cambiar versión de idioma

Cuando un usuario haya iniciado sesión tiene que cambiar al diálogo de cierre de sesión para cambiar el idioma (See « Cambiar la versión de idioma del usuario que ha iniciado sesión»).

  1. Seleccionar botón.
  2. Seleccionar el idioma deseado. El programa se adapta automáticamente al idioma elegido.
Cambiar Pin
  • Seleccionar el botón Cambiar PIN.
  • Dé el código PIN anterior y confirme con OK.
  • Dé el código PIN nuevo y confirme con OK.
  • Vuelva a dar el código PIN nuevo y confirme con OK.

Si los datos indicados son correctos puede iniciar sesión en el programa con el código PIN nuevo.
