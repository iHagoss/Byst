Importar pieza a doblar

En este registro pueden importarse piezas a doblar en el menú Home. La visualización puede filtrarse indicando el nombre de pieza en la línea de entradas.

Tipo de archivo
  • Todos los formatos de archivos conocidos

    • Con esta función puede importarse una pieza a doblar de una memoria de datos que se haya creado en formato .bpx o .bendpart.

  • Archivos BPX

    • Con esta función puede importarse una pieza a doblar de una memoria de datos que se haya creado con MMC ByBend o BySoft 6.x.

  • Archivos Bendpart

    • Con esta función puede importarse una pieza a doblar de una memoria de datos que se haya creado con el programa CAD/CAM BySoft 7.

  • Archivos DXF

    • Con esta función puede importarse una pieza a doblar en formato DXF con informaciones de doblado almacenada en una memoria de datos.

Clasificar según
  • Nombre

    • Clasifica las piezas a doblar según el nombre de pieza.

  • Fecha

    • Clasifica las piezas a doblar según la fecha de creación.

  • Tamaño

    • Clasifica las piezas a doblar según el tamaño de archivo.

  • Ascendente

    • Comenzando por el principio del alfabeto.

  • Ascendente

    • Comenzando por el final del alfabeto.

Importar

Importa las piezas a doblar seleccionadas en el conjunto de piezas del programa. Sólo puede importarse una pieza cada vez.

Conflictos al importar

Se muestran los conflictos de nombres que puedan surgir al importar automáticamente archivos *.bendpart.

Existen las siguientes posibles soluciones:

  • Suprimir el archivo original. El archivo no se importa.
  • Importar archivo y sobrescribir la pieza a doblar existente del mismo nombre.

    • La pieza ya cargada se descarga automáticamente y se suprime. Luego se importa la pieza nueva y se carga automáticamente con el mismo nombre.

    Attention! Con ello se pierden todos los cambios hechos en tecnología, correcciones, etc. de la pieza cargada.

Poner favorito

Memoriza la ruta seleccionada como favorito. En este registro se muestra siempre dicha ruta como ruta estándar.

Tip: La función Favoritos hace más fácil encontrar las piezas a doblar que quieran importarse y que estén memorizadas en una ruta estándar.
