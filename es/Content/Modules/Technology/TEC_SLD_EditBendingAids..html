En esta caja de diálogo puede cambiarse la posición de los dispositivos auxiliares para doblar, o ponerse de nuevo, para el proceso de doblado seleccionado. Las indicaciones no válidas se marcan con un símbolo de error .

Las opciones de cada dispositivo auxiliar para doblar se muestran en una sección propia. Con las teclas de flecha puede navegarse entre los diferentes dispositivos auxiliares de doblado.

Hay dos tipos de dispositivos auxiliares para doblar:

  • Dispositivos auxiliares para doblar posteriores: Pueden desplazarse automáticamente en el eje W a lo largo del alojamiento de herramienta.
  • Dispositivo auxiliar para doblar delantero: Por razones de seguridad no pueden desplazarse automáticamente en el eje W a lo largo del alojamiento de herramienta. El usuario tiene que ajustar manualmente la posición para la correspondiente estación de doblado.

Dispositivo auxiliar para doblar delante / atrás

En esta sección pueden ajustarse por separado las opciones para cada dispositivo auxiliar para doblar delantero o trasero. La cantidad de dispositivos auxiliares para doblar disponibles se muestra con un número.

Utilización
  • Colocado

    • Con ese atributo se han colocado en horizontal todos los dispositivos auxiliares para doblar disponibles si ello lo permite la técnica de doblado. Durante el proceso de doblado, el dispositivo auxiliar para doblar no realiza ningún movimiento giratorio de apoyo en paralelo al proceso de doblado.

    NOTE: Los dispositivos auxiliares para doblar delanteros tienen que posicionarse manualmente en la máquina por motivos de seguridad.

    Tip: Este atributo puede usarse para utilizar el dispositivo auxiliar para doblar como soporte de chapa.

  • Estacionado

    • Con este atributo se desplazan todos los dispositivos auxiliares para doblar disponibles a la posición de estacionamiento. El dispositivo auxiliar para doblar permanece en la posición de estacionamiento durante el proceso actual hasta que se desactive manualmente el atributo Estacionar a través de la función Editar en el menú Dispositivos auxiliares de doblar o se elija otra utilización.

  • Activado

    • Con este atributo se asigna automáticamente un dispositivo auxiliar para doblar a todos los procesos de doblado siempre y cuando ello sea técnicamente posible. Se tienen en cuenta todos los dispositivos auxiliares para doblar disponibles. Durante el proceso de doblado, el dispositivo auxiliar para doblar realiza un movimiento giratorio de apoyo en paralelo al proceso de doblado.

Posición H

Con esta opción puede adaptarse la altura del dispositivo auxiliar de doblado en el correspondiente proceso de doblado.

Tip: Utilizar esta opción para optimizar la altura de dispositivos auxiliares de doblado si el autómata no encuentra ninguna solución aprovechable.

Posición W

Con esta opción se ajusta la distancia del punto de referencia del correspondiente dispositivo auxiliar para doblar respecto al borde exterior izquierdo de la mesa de prensa. El posicionamiento puede hacerse de dos formas.

  • Haciendo la entrada de la posición en el campo Posición W.
  • Desplazando el regulador de desplazamiento.

Sincronización para dispositivos auxiliares de doblar activos

En esta sección puede ajustarse el movimiento sincronizado del dispositivo auxiliar para doblar activo con el movimiento de retroceso del portapunzón después de doblar.

Aplicación:

  • Para evitar que después de doblar piezas pesadas o grandes éstas resbalen sobre la herramienta inferior hacia la parte de atrás de la máquina.
  • Para evitar que la pieza a doblar se caiga del dispositivo auxiliar para doblar trasero sobre los talones de tope.
Hasta ángulo (α)

Con esta opción se determina el ángulo α hasta el que los dispositivos auxiliares de doblar (A) van sincrónicamente con el movimiento de portapunzón con velocidad reducida al retroceder después del proceso de doblado.

Info: La sincronización no se realiza si se ha indicado el ángulo máximo en el campo de entradas. El dispositivo auxiliar para doblar y el portapunzón se desplazan a velocidad normal hasta la posición de partida.
