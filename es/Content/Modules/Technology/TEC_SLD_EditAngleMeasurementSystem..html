Editar

En esta caja de diálogo puede cambiarse la posición del sistema de medición de ángulo, o ponerse una nueva, para el proceso de doblado seleccionado. Las indicaciones no válidas se marcan con un símbolo de error .

Función

En este campo se ajustan las funciones del sistema de medición de ángulo.

Modo

Con esta opción se conecta y desconecta el sistema de medición de ángulo según determinados criterios.

  • Conectar

    • Conecta el sistema de medición de ángulo para todos los procesos de doblado.

  • Primera conectada

    • Conecta la medición de los ángulos para el primer proceso de doblado respectivo con ángulos de doblado idénticos.

  • Desactivado

    • Desconecta el sistema de medición de ángulo. Sólo pueden ajustarse los ejes del sistema de medición de ángulo.

  • Adoptar

    • Adopta los valores medidos del proceso de doblado indicado en el campo Adoptar del proceso.

Adoptar del proceso

Con esta opción se adoptan los valores medidos en el proceso de doblado seleccionado.

Sensor

Los sensores seleccionados se utilizan para medir los ángulos. Ambos sensores deberían utilizarse de forma estándar porque con ello se logra mayor precisión.

Cada enésima pieza a doblar

Cuando haya una serie de piezas a doblar idénticas puede indicarse aquí tras cuántas piezas producidas debe comprobar el sistema de medición de ángulo otro pieza.

Ejes

En este campo se coloca el sistema de medición de ángulo a lo largo del alojamiento de herramienta.

Utilización

Con esta opción se posiciona el sistema de medición de ángulo o se le lleva a la posición de estacionamiento. Estando activada la opción Estacionado se desactivan todas las demás opciones del sistema de medición de ángulo.

Posición LZB/F

Con esta opción puede posicionarse el sistema de medición de ángulo en la correspondiente estación de doblado del proceso de doblado seleccionado.

  • B = Atrás
  • F = Delante
Desplazar

Con esta opción puede desplazarse el sistema de medición de ángulo con la mano pulsando y arrastrando a la posición deseada en la vista 3D (See «Manejar la pantalla táctil»).

Compensación de la recuperación elástica

En este campo se ajusta la corrección para la recuperación elástica del material durante el doblado. Sólo está activa si el modo de la función está conectado.

Modo

La medición de recuperación elástica se necesita para compensar la recuperación elástica del material durante el doblado.

  • Medición conectada

    • El ángulo de recuperación elástica se determina automáticamente.

  • Base de datos

    • Se utiliza el ángulo de recuperación elástica de la base de datos.

  • Manual

    • El ángulo de recuperación elástica tiene que indicarse manualmente a través de la opción Ángulo.

Ángulo esperado

Determina el ángulo esperado en la medición de la compensación de recuperación elástica. Se emite un mensaje de error si el ángulo sale del área indicada.

  • Base de datos

    • El ángulo esperado se lee de la base de datos.

  • Manual

    • El ángulo esperado puede indicarse en el campo ubicado por debajo.

Ángulo

En el caso de que se haya seleccionado el atributo Manual en la opción Ángulo esperado entonces hay que indicar en este campo el ángulo para la compensación de recuperación elástica.
