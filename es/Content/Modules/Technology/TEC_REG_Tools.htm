Herramientas

En este registro se determinan las herramientas de doblado con las que deban hacerse las dobladuras. Las herramientas de doblado pueden asignarse manual o automáticamente.

El proceso de doblado actualmente seleccionado se muestra como línea de doblado verde en la visualización 2D. Cada proceso de doblado puede tener asignada una operación de doblado o varias.

El proceso de doblado se selecciona haciendo clic sobre una línea de doblado.

Visualización de herramienta para procesos de doblado

Debajo de la visualización 2D se muestra tabularmente la asignación de herramientas a cada uno de los procesos de doblado. Haciendo clic sobre el proceso de doblado se abre el diálogo Editar.

Si hay pendientes avisos o errores sobre la herramienta de doblado se indican con un símbolo detrás de la herramienta de doblado. Moviendo el ratón sobre el símbolo de aviso se muestran informaciones sobre el aviso.

Tip: El aviso o error puede confirmarse en el diálogo Editar en el caso de que pueda pasarse por alto para el proceso de doblado (See «Botones»).

NOTE: En el caso de que no se confirme el mensaje hay que elegir otra herramienta adecuada para el proceso de doblado.

Muestra el método de doblado con el que se dobla el proceso.

Operación de doblado

Muestra el tipo de doblado en el proceso de doblado.

  • Doblado normal El doblado normal es el método usual de hacer un doblado. El doblado queda determinado por el ángulo de doblado y un ángulo de predoblado opcional.
  • Dobladura plana Este tipo de doblado pliega la chapa con un ángulo de doblado de 180° dejándola plana. El proceso de doblado se ejecuta en los dos pasos consecutivos «Predoblado» y «Dobladura plana». Estos pasos pueden ejecutarse también independientemente.
  • Predoblado Este tipo de doblado hace una dobladura en dos pasos. El doblado se dobla primero con un ángulo de predoblado y posteriormente se acaba de doblar con el ángulo final. Utilice el predoblado en piezas complejas a doblar para evitar colisiones durante el proceso de doblado.
  • Doblado auxiliar El doblado auxiliar se utiliza en determinadas formas de doblado de forma provisional, que luego se desdobla, para posibilitar dar la forma final a una pieza a doblar.
  • Doblado El doblado es un elemento de la programación de perfiles y designa un cambio de forma en la pieza a mecanizar. Es posible asignar varios atributos a un doblado. Longitud de lado, ángulo de doblado, radio de doblado (optativo), ángulo de predoblado (optativo), atributos adicionales (optativo).
