En esta caja de diálogo pueden hacerse ajustes de compensación para cada proceso de doblado.

  • Ventajas: Aumenta la precisión de los correspondientes procesos de doblado.
  • Inconveniente: Con ello puede aumentar considerablemente el tiempo de doblado de una pieza.

NOTE: Dichos ajustes vuelven a sobrescribirse tras la asignación automática de la compensación.

Dichas opciones están disponibles para las siguientes máquinas, siempre y cuando se haya activado la opción correspondiente en el Administrador:

  • Xpert: El bombeado puede ejecutarse de forma dinámica y estática.
  • Xact: El bombeado sólo puede ejecutarse de forma estática.

Compensación del grosor de chapa

Los ajustes en este sector determinan la forma en que se compensan las fluctuaciones del grosor de chapa.

Modo
  • Desactivado

    • La compensación del grosor de chapa está desconectada para el proceso seleccionado.

  • Conectar

    • La compensación del grosor de chapa está conectada para el proceso seleccionado.

  • Adoptar

    • La compensación del grosor de chapa adopta el valor medido del proceso que se haya indicado en la opción Adoptar del proceso.

Adoptar del proceso

Determina el proceso del que se adopta el valor de compensación medido para el proceso seleccionado.

NOTE: Sólo se pueden seleccionar procesos cuyo modo esté ajustado a Conectado.

Comportamiento del portapunzón en el punto de sujeción
  • Estándar

    • Este es el ajuste estándar para el comportamiento del portapunzón.

    El portapunzón se para en el punto de sujeción si se ha programado un retroceso y ejecuta el retroceso de los topes antes del doblado.

    ¡Si se ha programado un retroceso puede fallar la medición del grosor de chapa para el proceso seleccionado! Utilice en dicho caso el atributo No parar nunca.

  • No parar nunca

    • El portapunzón no se para en el punto de sujeción si se ha programado un retroceso para garantizar la medición del grosor de chapa para la compensación. ¡El retroceso se realiza a pesar de ello!

  • CAUTION:

    ¡Peligro de colisión!

    No utilice nunca esta opción si hay lados doblados (A) debajo del talón de tope (B). El lado puede colisionar con el talón de tope y doblar o apartar la pieza hacia atrás.

Bombeado Es un sistema para compensar las desviaciones provocadas por el combado de la barra superior e inferior durante el proceso de doblado.

Attention! Utilizando la función Automático en las máquinas Xpert se vuelve a poner la opción Bombeado estático al ajuste estándar Bombeado dinámico.

Modo
  • Estático

    • La presión se calcula y ajusta según la fuerza de presión.

    La presión ya está activa cuando el portapunzón sale del PMS y se mantiene hasta que finalice el doblado.

  • Dinámico

    • En este caso, el CNC ajusta la presión automáticamente. Cuanto mayor sea la fuerza de presión tanta más presión se genera para el bombeado. La presión del bombeado se mantiene desde el punto de sujeción hasta que finalice el doblado.
