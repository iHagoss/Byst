Funciones

Con las funciones de la barra de funciones pueden crearse planes de equipamiento para procesos de doblado.

Puede disponer de las siguientes funciones:

Funciones Significado
Tecnología Con esta función se determina toda la tecnología de doblado y se le asigna automáticamente a la pieza a doblar.
Automático Con esta función se posiciona automáticamente el sistema de medición de ángulo (LAMS Véase el sistema de medición de ángulo) en la estación de doblado de cada uno de los procesos.
Optimizar

Con esta función se aplica el sistema de medición de ángulo automáticamente a todos los procesos de doblado que todavía no tengan asignado ningún sistema de medición.
Comprobar Con esta función se comprueba si el estado del sistema de medición de ángulo es válido para los procesos.
Ajustes En esta caja de diálogo se determinan los criterios para el cálculo automático del sistema de medición de ángulo.
Editar En esta caja de diálogo puede cambiarse la posición del sistema de medición de ángulo, o ponerse una nueva, para el proceso de doblado seleccionado.
Reposicionar Con esta opción vuelven a calcularse las posiciones del sistema de medición de ángulo y se reposicionan si es necesario.
