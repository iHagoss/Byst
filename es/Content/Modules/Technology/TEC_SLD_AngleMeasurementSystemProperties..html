Ajustes

En esta caja de diálogo se determinan los criterios para el cálculo automático del sistema de medición de ángulo.

Función

Utilización

Con esta opción se define la posición básica del sistema de medición de ángulo.

  • Colocado - Todas las mediciones Off

    • El sistema de medición de ángulo se desactiva para todos los procesos de doblado y se posiciona en la estación de doblado actual.

  • Colocado - Todas las mediciones On

    • El sistema de medición de ángulo se activa para todos los procesos de doblado y se posiciona en la estación de doblado actual.

  • Colocado - Primer medición por ángulo de doblado On

    • El sistema de medición de ángulo se activa y se posiciona en la estación activa. En el caso de que se doblen sucesivamente varios ángulos de doblado idénticos, entonces se mide sólo el primer ángulo de doblado hasta que se doble un ángulo nuevo.

  • Estacionado

    • Con este atributo se desplaza el sistema de medición de ángulo a la posición de estacionamiento. El sistema de medición de ángulo permanece en la posición de estacionamiento durante el proceso actual hasta que se desactive manualmente el atributo Estacionar a través de la función Editar en la opción Utilización o se elija otra utilización

Info: Un servomotor posiciona automáticamente el sistema de medición de ángulo.

Compensación de la recuperación elástica

Modo

La medición de recuperación elástica se necesita para compensar la recuperación elástica del material durante el doblado.

  • Medición conectada

    • El ángulo de recuperación elástica se determina automáticamente.

  • Base de datos

    • Se utiliza el ángulo de recuperación elástica de la base de datos.

  • Manual

    • El ángulo de recuperación elástica tiene que indicarse manualmente a través de la opción Ángulo.

Ángulo esperado

Determina el ángulo esperado en la medición de la compensación de recuperación elástica. Se emite un mensaje de error si el ángulo sale del área indicada.

  • Base de datos

    • El ángulo esperado se lee de la base de datos.

  • Manual

    • El ángulo esperado puede indicarse en el campo ubicado por debajo.

Ángulo

En el caso de que se haya seleccionado el atributo Manual en la opción Ángulo esperado entonces hay que indicar en este campo el ángulo para la compensación de recuperación elástica.
