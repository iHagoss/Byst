Bending process

On this tab you can correct the parameters for the bending process.

Crowning factor

Crowning A system for compensating for deviations that can be traced to deflection of the top and bottom tool during the bending process. can be adjusted between the minimum value of 0 and the maximum value of 10.

  • <1 = reduces crowning
  • >1 = increases crowning
Cycle counter

In this input box you can correct the cycle counter value.

  • 0 = minimum (skip process)
  • 10,000 = maximum
Thickness of protective insert

In this input box you can change the thickness of the protective insert.

Control unit

In this dialog you can choose between one or two pedals. Then press the button for the function for which you wish to apply the settings you have entered.

The following functions are available:

  • Process

    • Entered value is adopted for the current process.

  • Parallel

    • Entered value is adopted for all bends that have the same angle on the same section

  • Station A station consists of one or more segments of a top tool and a bottom tool.

    • Entered value is adopted for all processes in the current station

  • Part

    • Entered value is adopted for all processes of the workpiece
