Properties

This dialog is where you enter bending technology data for the profile to be bent. The dialog is divided into the following tabs:

Profile

On this tab you can select or input profile details. The details apply to all bends for the profile.

If you need to use different tools, bending methods or bend lengths for separate bends, you can define them on the Bends tab.

  • System of units

    • Defines the system of units (mm/inches).

  • Material

    • Defines the material. You can select any material entered in the Administrator.

  • Sheet thickness

    • Defines the sheet thickness.

  • Bend The bend is an element of the profile programming and designates a change in shape of the workpiece. A number of attributes can be assigned to a bend: side length, bend angle, bend radius (optional), initial-bend angle (optional), additional attributes (optional). A bend can consist of several bending processes. length

    • Defines the length of the bend line.

  • Machine

    • Defines the bending machine. You can select any machine entered in the Administrator.

  • Bending method 3-point bending, air bending and coining are referred to as bending methods.

    • Defines the air bending, 3-point bending 3-point bending achieves a repeatable and highly accurate angle setting using the V-shaped system. The bend angle is determined by the two entry edges, the bottom tool and the channel floor of the bottom tool. The 3 points lie on the same side of the sheet. The angle is set using height-adjustable stops in the bottom tool. The hydro cushion compensates for deformation by the ram. Each of the segmented top tools works autonomously against the hydro cushion and is protected against overload. The bending process is not complete until the material lies on the channel floor of the bottom tool, and the top tool is submerged in the hydro cushion. or coining bending method depending on the bending machine being used.

  • Top tool

    • Defines the top tool for the bending station. You can select any top tool entered in the Administrator.

  • Bottom tools

    • Defines the bottom tool for the bending station. You can select any bottom tool entered in the Administrator.

  • Top/bottom tool details

    • This dialog lists further attributes of the selected top and bottom tools.

Section

For each section, you can personally select the bend length or adopt the length from the profile. The length of the blank part is also displayed for each section.

Bend

You can define details for the selected bend on this tab. The details only apply to the current bend.

  • Angle

    • Defines the angle of the bend.

  • Length

    • Defines the length of the selected side.

  • Bend length

    • Defines the length of the bend line. You can enter a custom length or use the length of the section.

    Info: The option same as section copies the bend length from the Section tab.

  • Bending method

    • Defines the air bending, 3-point bending or coining bending method depending on the bending machine being used.

    Info: The Default setting copies the bending method from the Profile tab.

  • Top tool

    • Defines the top tool for the bending station. You can select any top tool entered in the Administrator.

    Info: The Default setting copies the top tool from the Profile tab.

  • Bottom tool

    • Defines the bottom tool for the bending station. You can select any bottom tool entered in the Administrator.

    Info: The Default setting copies the bottom tool from the Profile tab.

  • Radius tools...

    • Press the button to choose a suitable tool from a selection based on a predefined radius.

    You cannot enter any inside radius you wish, however, because this cannot be achieved with the tools available.

  • Stepped radius A stepped radius bend is made when the angle and radius of a bent side is achieved by making a number of individual bends.

    • One stepped bend can be defined per bend. Selecting the Stepped radius option opens a dialog to define a stepped bend.

    Type

    The edges and surfaces of a stepped bend do not run perfectly parallel to the actual bend radius. This option defines whether the bend radius runs along the inside (A) or outside (B) of the stepped bend. By default, the attribute Tangential inside is used.

    Fig.: Tangential inside/outside

    Inside radius

    Defines the inside radius (R) of a stepped bend.

    Number of bends

    This option defines the number of bends with which to bend the radius. Press Maximum to enter the maximum possible number of bends in Number of bends.

    The greater the number of bends, the more exact the bend radius and the more complex the bend.

    Info: The maximum number of bends depends on the tool settings.

  • Radius

    • If the Automatic option is enabled, the bend radius is determined by the bending tool selected. If the option is not enabled, you can enter a custom number.

  • Bend allowance The reduction in the side length for bends made on the bending part is called a bend allowance. Bending causes the part to be stretched in a longitudinal direction. Therefore the length of the side of a bend must be reduced again by this amount in the cutting part. The bend allowance, i.e. shortening of the side length, compensates for this stretching.

    • If the Automatic option is enabled, the bend allowance is determined by the bending tool selected. If the option is not enabled, you can enter a custom number.

General

This is where the descriptions and extra information about the created profile are presented. The information lines and description can be entered before saving the profile for the first time.

Info: A newly created profile must be saved under a name, however, before the name, creation date and revision information can be displayed.
