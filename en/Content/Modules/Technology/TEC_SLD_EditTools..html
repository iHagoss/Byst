Edit

In this dialog box, you can assign to the currently selected bending process a top tool and bottom tool for the selected Bending method 3-point bending, air bending and coining are referred to as bending methods.. The tools are defined using the appropriate selection boxes. For the each tool, the program displays the contour and respective attributes.

The Apply button runs a check of the chosen tools and applies them to the bending process, the current bend or the entire bending part. If the tools are not suitable for the bending process, the last valid tools are used.

Top / bottom tool

Use the tabs for the top and bottom tool to define the appropriate bending tools.

  • Selection box for tools

  • Show all tools

    • This option shows all tools that have a different bend allowance. Enable this option if you wish to select tools that have a different bend allowance.

    NOTE:

    If a bend allowance changes, the geometry of the part is not corrected accordingly. Hence changing the bend allowance results in a change in the side length of the part.

    This option should be disabled by default in order to ensure that only tools having the same bend allowance are used.

  • Tool extension

    • Use these tabs to define tool adapters for the top and bottom station.

    Adapters can be used for the following applications:

    • As an extension of the bending tool to avoid collisions with the machine beam.
    • As tool adapters to use bending tools from a different manufacturer.
Bending method

In this section you can select the bending and hemming method for the selected bending process.

Bending method

  • Air
  • Coining Open-die and closed-die coining is the most precise bending method. The shape and the bend angle are set by the geometry of the top and bottom tools. The extremely high pressing forces that are required increase the wear on the press brake.

Hemming This bend type bends the sheet over flat with a bend angle of 180°. The bending action is completed in two consecutive steps of «Pre-bend» and Hem. These steps may also be performed separately from one another. method

If the selected bend is of the hem type, you can select the method in the Hemming selection dialog.

  • Standard embossing: Hemming is performed with normal pressing force. After bending, a small air gap remains along the bend line (A).
  • Strong embossing: Hemming is performed with increased pressing force. After bending, there is practically no air gap along the bend line (A).

Fig.: Hemming methods

NOTE: In order to enable the dialog for hemming, select the Coining method in the Bending method selection dialog or, for Hämmerle bending machines, additionally 3-point.

Buttons
  • Apply

    • Assigns the top and bottom tool to the selected bending process.

  • Clear conflicts

    • You can clear warnings and errors for the process, bend, station or part.

  • Warnings

    • This function acknowledges and clears the warning for the selected bending process. The warning is displayed as a symbol after the tool name.

    Tip: Use this function for warnings that can be ignored for the selected bending process. Acknowledging the warning tells the system that the tool can be used for the bending process.

  • Error

    • This function clears errors from allocated tools.

    Info: This function is only enabled if the pending error can be cleared.
