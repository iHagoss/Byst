Settings

In this dialog box you can specify the criteria for employing the angle measurement system and its automated functions.

Function

Use

This option defines the basic setting for the angle measurement system.

  • Positioned - all measurements, off

    • The angle measurement system is disabled for all bending processes and positioned on the current bending station.

  • Positioned - all measurements, on

    • The angle measurement system is enabled for all bending processes and positioned on the current bending station.

  • Positioned - first bend angle measured per angle

    • The angle measurement system is enabled and positioned on the current bending station. If several identical bending angles are bent successively, only the first bend angle is ever measured until a new angle is bent.

  • Parked

    • You use this attribute to move the Angle measurement system The angle measurement system minimizes the material-dependent variations in the bend angle during the bending process. This ensures that excellent angle precision is maintained over the entire length of the part. into the parked position. The angle measurement system remains in the parked position in the current process until the Parked attribute is manually disabled using the Edit function in the Usage option, or until another use has been selected.

Info: The angle measurement system is positioned automatically by a servomotor.

Every n'th bending part

For a series of identical bending parts, you can enter the number of produced parts after which you want the angle measurement system to check another part.

Spring-back compensation

Mode

The spring-back measurement is needed to correct for the spring-back of the material during bending.

  • Measurement On

    • The spring-back angle is measured automatically.

  • Database

    • The spring-back angle from the database is used.

  • Manual

    • The spring-back angle must be entered manually via the Angle option.

Expected angle

Defines the expected angle when measuring spring-back compensation. If the angle is no longer in the defined range, an error message is shown.

  • Database

    • The expected angle is read from the database.

  • Manual

    • You can enter the expected angle in the box below.

Angle

If the Manual attribute is selected in the Expected angle option, the spring-back compensation angle must be entered in this box.

Optimize

LZ position

If possible, the program attempts to use the same LAMS See Angle measurement system positions in the LZ direction.

Position window

Defines the maximum distance of the axis in the LZ direction which the angle measurement system may be moved automatically.
