Edit segment

In this dialog you can add all available tool segments to a station.

  • Segment A segment is a bending tool that is used as a bending station or part of a bending station.

    • This option adds the selected segment to the bending station.

  • Blank segment

    • This option adds an empty segment of any length to the station.

Fig.: Segment types on a station plan

Key:

A Normal segment
B Segment with left/right horn
C Station A station consists of one or more segments of a top tool and a bottom tool. with blank segment
