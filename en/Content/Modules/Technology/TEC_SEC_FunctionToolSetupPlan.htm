Functions

Use the functions on this toolbar to create tool setup plans for the bending processes.

The following functions are available:

Functions Meaning
Technology This function is used to set the entire bending technology and automatically assign it to the bending part.
Automatic This function automatically creates the tool setup plan according to the criteria specified in the Automatic settings dialog box.
Complete This function calculates all bending stations that have not yet been set on the tool setup plans. Any bending stations that have already been set are retained and are not recalculated.
Import This dialog is used to import the tool setup plan for an existing bending part into the current bending part.
Tool detection In this dialog you configure the options for automatic detection of tools.
Check This function checks whether all the processes have been assigned a bending station.
Settings This dialog is used to specify the criteria for automatic computation of the tool setup plan.
Edit In this dialog you can edit an existing tool setup plan or create a new one.
Assign In this dialog box you can assign new bending processes to a bending station or change the assignment of bending processes to a bending station.
Center This function centers the bending station on the station plan.
