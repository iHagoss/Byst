Tool setup plan

The tool setup plan for the bending part is created on this tab. A tool setup plan can consist of one or more station plans. A check function lets you check the validity of all station plans you have created.

The view is divided into a 3D and a 2D view. You can use various bars in the view to edit the bending part.

2D view

This view displays the station plan and all the stations. If more than one station plan is being used, you can switch between the various station plans using the two arrow keys.

  • The program displays the length in mm for each tool.
  • The station number between the top and bottom tool is displayed for each station.
