Offset

In this dialog box you can define the position of the part on the bending station. Specifying different positions is used to avoid collisions when computing the bends.

Attention! If the position on the bending station is changed when backgauges have already been set, you must run the Reposition function on the Backgauges tab for the process (See «Reposition»).

You have the following options for positioning a bending part on the station:

  • Left

    • Moves the bending part to the left-hand end of the station. Use this option to prevent tongues colliding with the station during bending.

  • Centered

    • Centers the bending part in the station. Use this option as the standard option for air bending.

  • Right

    • Moves the bending part to the right-hand end of the station. Use this option to prevent tongues colliding with the station during bending.

  • User-defined

    • Moves the bending part to a user-definable position. Use this option in the following situations.

    • On stations containing empty segments to adjust the workpiece position to suit the station.
    • For 3-point bending 3-point bending achieves a repeatable and highly accurate angle setting using the V-shaped system. The bend angle is determined by the two entry edges, the bottom tool and the channel floor of the bottom tool. The 3 points lie on the same side of the sheet. The angle is set using height-adjustable stops in the bottom tool. The hydro cushion compensates for deformation by the ram. Each of the segmented top tools works autonomously against the hydro cushion and is protected against overload. The bending process is not complete until the material lies on the channel floor of the bottom tool, and the top tool is submerged in the hydro cushion. in order to comply with the 75% rule If «3-point» is selected as the bending method, the load on a tool segment must equal 75% minimum. If this minimum value is not met, the hydro cushion on the 3-point bending machine can sustain permanent damage. If the load cannot be increased to at least 75% owing to the use of another segment, the bending method must be set to Air bending..
  • Process

    • Assigns the selected position only to the current process.

  • Station A station consists of one or more segments of a top tool and a bottom tool.

    • Assigns the selected position to all processes on the station.
