Export to

Exports the current bending part to a custom folder. You can enter the path at the bottom of the dialog.

  • File name

    • You can enter the name of the bending part in this box. By default, the program uses the name of the current bending part.

  • Export to

    • Exports the bending part to the specified folder.

  • File type

    • Defines the format in which to save the bending part.

  • Controller version

    • Defines the controller version in which to save the bending part. This option is only available for the Bendpart file file type.
