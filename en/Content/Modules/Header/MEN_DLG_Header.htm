Elements in the header

The header displays the main functions and menus. The header is displayed in all views.

Info: Tap on the text box or the icon to display additional menus. The additional menus are automatically hidden after a short time.

Displaying help about functions in the header

To display the help in the Help window, tap the function in the header and drag your finger down slightly (See «Swiping»).

Bending jobs

Switches to bending jobs view to automatically process multiple bending parts.

Home

Switches to Home view to edit the properties of the bending part.

Edit technology

Switches to Technology view to edit the bending technology of the bending part.

Tool setup plan

Switches to Tool setup plan view to display quick information about the tool setup plans for the bending part.

Produce

Switches to Produce view to produce the bending part.

Programs & functions

Opens the menu to launch external programs and additional functions.

Text box

Shows the name of the current view.

Top tool clamp

Opens or closes the top tool clamp If the LED is green, the clamp is closed.

Bottom tool clamp

Opens or closes the bottom tool clamp If the LED is green, the clamp is closed.

Hydraulic motor

Turns the hydraulic motor on or off. If the LED is green, the hydraulic motor is switched on.

Power saving button

The function is enabled if the LED is blue. The controller switches off the main motor when the machine is not in use.

Light

This menu switches the front and rear machine light on or off.

System messages

Opens the dialog to display system and error messages.

Help functions

Opens the menu to display various help functions.

Close

Opens the menu to change user or close the program.
