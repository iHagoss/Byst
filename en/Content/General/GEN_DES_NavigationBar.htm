Navigation bar

The Navigation bars can be used to simulate the bending process and the movement of machine components. Different icons are displayed in the bar depending on the menu.

Bending process technology

This Navigation bar is available in every Technology menu except the Tools menu. This area controls the display and sequence of the bending processes.

  • Jumps to the next process that contains an error
  • Switches to the first process
  • Switches to the previous process
  • Switches to the next process
  • Switches to the last process
Produce bending process

This navigation bar controls the display and sequence of the bending processes.

  • Switches to the first process
  • Switches to the previous process
  • Switches between single-step and automatic mode
  • Switches to the next process
  • Switches to the last process
Simulation

This navigation bar controls simulation of the bending process. The navigation bar is only displayed in the Simulation menu.

  • Starts the simulation
  • Interrupts the simulation
  • Stops the simulation
  • Defines the speed of the simulation
  • Enables simulation of the movement of backgauges, bending aids and the angle measurement system.
