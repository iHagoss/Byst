Displaying the Online Help

The online help facility is an integral part of the software. Depending on the language version of the operating system, the program and the online help are displayed in the appropriate language version.

Help display options

There are two ways of displaying the help from the main menu in this program.

Symbol Meaning
Complete help

Displays the help system in a separate help system. The following navigational aids are available for navigating the help system:

  • Contents
  • Index
  • Glossary
  • Full-text search
Context-sensitive help

Displays the help system in an integrated Help dialog in the program. The contents are displayed based on the context of the selected software element.

Press the pin icon in the Help dialog to show or hide the context function.

  • Context-sensitive help enabled
  • Context-sensitive help disabled
