Menus and display elements

Menus, dialogs and functions almost all have icons assigned to them. This concept facilitates intuitive use of the program and creates a tidy user interface.

Info: When you show or hide dialogs, the displays scale dynamically to fit the interface. This allows you to show additional dialogs without them covering up existing views.

Header

The icons in the header are used to invoke all sections and modules of the program (See «Elements in the header»).

Select the text area (A) of the header to show additional menu items. When no button is pressed, the program hides these menu items again.

Menu bar

If you select the Edit technology function, the menu bar containing buttons for each of the Technology menus is displayed on the left of the screen.

 

View window

The view window contains the main view and also additional dialog windows depending on the selected function. Depending on the menu, the dialog windows display the bending processes and views showing the bending sequence and station plan.

Toolbar

The buttons in the toolbar can be used to start actions or select functions for the view concerned. If no functions are available in the current view then the toolbar is not displayed.

 

Message list

The message list displays all pending messages, indicating the status and origin.

Slide dialogs

These dialogs automatically slide into the interface from right to left. The original dialog is hidden. Tap the dialog header (A) to hide it again and show the previous dialog again.

Tabs

Tabs are used to present the various sections of the software in a clutter-free manner. The user taps on the appropriate tab to display the dialogs. Each tab is marked by a certain icon or text (A).

 

On-screen keyboard

You can use this keyboard dialog to input custom values in a box. The dialog is displayed as soon as you select a box that allows custom values.

Numerical input dialog

This dialog is used to input numerical values in a box. The dialog is displayed as soon as you select a box that allows numerical values.

Selection dialog

This dialog is used to enter pre-defined values in a box. The dialog is displayed as soon as you select a box with pre-defined values.
