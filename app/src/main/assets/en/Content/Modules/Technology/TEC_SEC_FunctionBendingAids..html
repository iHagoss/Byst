Functions

Use the functions on this toolbar to create bending aids for the bending processes.

The following functions are available:

Functions Meaning
Technology This function is used to set the entire bending technology and automatically assign it to the bending part.
Automatic This function automatically positions the bending aids in the correct position on the bending part for each of the processes.
Check This function checks the bending aids for collisions with the bending aids themselves, with the bending part and with the machine.
Settings This function is used to specify the criteria for automatic computation of the required bending aids.
Edit In this dialog box you can change the position of the bending aids or set a new bending aid for the selected bending process.
Reposition This option recalculates the positions of all active bending aids for all processes, and repositions the bending aids as necessary.
