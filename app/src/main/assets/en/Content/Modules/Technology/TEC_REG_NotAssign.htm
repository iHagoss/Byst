Not assigned

This tab lists all the bending processes that have not been assigned to a bending station yet.

What each button does:

  • Automatic

    • Automatically assigns a bending station to all unassigned bending processes.

  • Assign

    • Assigns the bending station selected in the Tool setup plan section to the selected bending process. If no bending process is selected, the button is disabled.

Assigning unassigned bending processes to a bending station
  1. Select the bending station to assign to the processes in the Tool setup plan.
  2. On the Not assigned tab, select the bending process that you wish to assign to the new bending station.
  3. Press the Assign button. The bending process is reassigned to the bending station and moved to the Assigned tab.

Tip: You can also assign bending processes by selecting the bending process and dragging it over the appropriate bending station.

What each column means:

Process

A serial number indicates the sequence of the bending processes.

Top tool

Shows the top tool being used.

Bottom tool

Shows the bottom tool being used.

Bend The bend is an element of the profile programming and designates a change in shape of the workpiece. A number of attributes can be assigned to a bend: side length, bend angle, bend radius (optional), initial-bend angle (optional), additional attributes (optional). A bend can consist of several bending processes. length

Shows the bend length of the displayed bending process.
