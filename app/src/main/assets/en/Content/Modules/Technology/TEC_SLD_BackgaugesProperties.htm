Settings

This function is used to specify the criteria for automatic computation of the required backgauges for all bending processes.

  • Minimum edge length

    • This option sets the minimum length of an edge for which a backgauge can be selected. A default length of 10 mm is set. Only those numbers of the contact edges that meet this minimum length can be selected using the Contact edge The edge on the bending part that is pressed against a stop point on the backgauge finger for precise positioning. option in the Edit backgauges menu.

  • Retraction Retraction refers to the backwards movement of the backgauge arm that is required to avoid the backgauge finger colliding with the bending part. Retraction is performed once the bending part has been clamped between top and bottom tools. limit

    • This setting defines the lower limit for performing retraction. No retraction is performed below this setting and it is set to 0.

    Use:

    This setting allows you to accept minor collisions without having to retract. This is often used with thick sheets. The default minimum retraction is 0.2 mm (See «Retraction»).

Optimize

  • Z position

    • If possible, the program attempts to use the same backgauge positions in the Z direction.

  • Position window

    • Defines the maximum distance of the axis in the Z direction which the backgauges may be moved automatically.

  • End bend

    • For bending processes for which no backgauges are used, the X Position of the previous bending process is used.

    Bends without backgauges:

  • Auxiliary bend The auxiliary bend is temporarily inserted for certain bend shapes, and then bent back again, to allow the final shape of the bending part to be achieved.
  • Pre-bending This bend type creates a bend in two steps. The bend is first made with an initial-bend angle and then the finished bend is made later with a final angle. Use pre-bending for complex bending parts to avoid collisions during the bending process.
  • Hemming This bend type bends the sheet over flat with a bend angle of 180°. The bending action is completed in two consecutive steps of «Pre-bend» and Hem. These steps may also be performed separately from one another.
  • Parked position

    • This option prevents the unused backgauge from moving to Z parked position when using short backgauge edges. The program tries to use the Z backgauge positions from the previous bending process. If this is not possible, the backgauge that is not required is parked directly next to the backgauge being used.
