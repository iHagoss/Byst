Tools

On this tab you define the bending tools that you wish to use to produce the bends. The bending tools can be assigned manually or automatically.

The currently selected bending process is shown in the 2D view as a green bending line. One or more bending operations can be assigned to each bending process.

Clicking on a bend line to select the bending process.

Tool display for bending processes

Below the 2D view, a table displays the assignment of the tools to each of the bending processes. Click on the bending process to call up the Edit dialog.

If there are any pending warnings or errors for the bending tool being used, this is indicated by an icon next to the bending tool. Hover the mouse over the warning symbol to display details about the warning.

Tip: If you can ignore the warning or error for the bending process, you can clear them in the Edit dialog (See «Buttons»).

NOTE: If the message is not acknowledged, another tool suitable for the bending process must be selected.

What each column means:

Bending method 3-point bending, air bending and coining are referred to as bending methods.

Displays the bending method that is used to bend the process.

Bend The bend is an element of the profile programming and designates a change in shape of the workpiece. A number of attributes can be assigned to a bend: side length, bend angle, bend radius (optional), initial-bend angle (optional), additional attributes (optional). A bend can consist of several bending processes. type

Shows the type of bend in the bending process.

  • Normal bending Normal bending is the standard method for creating a bend. Bending is defined by the bend angle and an optional initial-bend angle.
  • Hemming This bend type bends the sheet over flat with a bend angle of 180°. The bending action is completed in two consecutive steps of «Pre-bend» and Hem. These steps may also be performed separately from one another.
  • Pre-bending This bend type creates a bend in two steps. The bend is first made with an initial-bend angle and then the finished bend is made later with a final angle. Use pre-bending for complex bending parts to avoid collisions during the bending process.
  • Auxiliary bend The auxiliary bend is temporarily inserted for certain bend shapes, and then bent back again, to allow the final shape of the bending part to be achieved.
  • Stepped radius A stepped radius bend is made when the angle and radius of a bent side is achieved by making a number of individual bends. bend

Angle

Shows the angle of the bend. An angle of 0 is only used for the final bend in a hemming operation.
