Settings

In this dialog you specify the options for selecting the tools and creating the bending sequence. An optimum bending sequence can be calculated by making suitable settings.

Press the Save as default button to apply the settings by default to the current part when using the Automatic function.

Tip: Click on the Automatic function to apply the settings to the bending part.

Settings

Check whether bending part will collide with itself

This option prevents the sides of a part from touching. This option should only be used in special cases as it seriously restricts the possible bending sequences or even excludes any bending sequences.

Tip: If sides are bent towards each other then these may touch along the edge. If this option is enabled, a collision warning is displayed even though the bend could be made without a problem. You will need to disable this option to prevent this.

Rotate top tool automatically

This option rotates the top tool in the tool holder through 180° when the number of valid bending sequences specified under the option Max. number of bending sequences cannot be found using the standard tool.

Use alternative tools

Enable this option to permit top tools that have different bend radii. If it is not possible to use a tool with the same bend radius, the program searches for a tool with a bend radius that is the next closest in size.

NOTE: If a bend allowance changes, the geometry of the part is not corrected accordingly. Hence changing the bend allowance results in a change in the side length of the part.

Info: This option can only be selected when the option Select top tool automatically is enabled.

Max. number of bending sequences

This option sets the maximum number of bending-sequence solutions that you wish the program to find for the parts. If this number is reached, the best solution is assigned to the part as the bending sequence.

Info: Increasing the number can lead to considerably longer computing times for complex bending parts.

Assess backgauges

This option determines the bending sequence on the basis of the better position of the rear backgauges.

Info: This option can only be selected if a number >1 has been set for the option Max. number of bending sequences.

Elastic tolerance

Attention! Only use this function for thin sheets (up to about 2mm). For thicker sheets and larger tolerance angles, the tool or the bending machine may be damaged if it collides with the bending part with too great a force.

If slight contact needs to be made with the top or bottom tool in order to produce a bend, then enter the tolerance value (in degrees) in the option field. You can enter a maximum tolerance angle of 15°. This tolerance value is then added to the target value of the bend angle when checking the angles. No collision message is issued inside this value.

NOTE: The deformation of the material when it collides with the top or bottom tool must remain within the elastic region of the material. Otherwise the material is permanently deformed!

Criteria

Position of center of gravity of part

This option specifies whether the center of gravity of the part is preferably in front of or behind the bending tool for the bending sequence.

Info:

Center of gravity, front: Remember for very long sides the high angular velocity during bending.

Center of gravity, rear: For very long sides, make sure that the bending part rests on the backgauge finger to prevent sagging of the side.

Fig.: Center of gravity of part in front of the bending tool

Bend length

This option specifies whether short or long sides of a part shall preferentially be bent first.

Fig.: Short bend length first

Center of part

This option specifies which bend shall preferentially be made first based on the distance from the center of the part.

Fig.: Those further from center of part first

Assessment criteria

This setting affects the weight attached to a criterion when calculating the bending sequence. The higher a criterion is placed, the more weight is given to this criterion in the calculation. You can use the arrow buttons to shift the weighting order of the criteria.

Info: The criteria correspond to the options above and the properties that have been set for them. The Sheet handling option means that optimum bending of the part is possible.
