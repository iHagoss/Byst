Functions

Use the functions on this toolbar to create backgauges for the bending processes.

The following functions are available:

Functions Meaning
Technology This function is used to set the entire bending technology and automatically assign it to the bending part.
Automatic This function positions the backgauges automatically on the bending edges of the individual processes, based on the criteria set in the Automatic settings dialog.
Complete This function automatically calculates all backgauges that have not yet been set. Any backgauges that have already been set are retained and are not recalculated.
Optimize This function attempts to reduce axis movement of the backgauges between processes. The optimization is performed based on the settings in the Automatic settings dialog.

Info: This option is enabled active is all backgauges have a valid status.

Check This function checks the backgauges for collisions with the backgauges themselves, with the bending part and with the machine.
Settings This function is used to specify the criteria for automatic computation of the required backgauges for all bending processes.
Edit This function is used to change the position of the backgauges or set a new backgauge for the selected bending process.
Reposition This function recalculates the axis positions of the backgauges for all the processes, repositioning them as necessary.
