Edit

In this dialog you can edit an existing tool setup plan or create a new one. The tool setup plan is displayed with all the stations and tool segments contained in the plan.

  • The top view shows the individual tool setup plans.
  • The bottom view displays the appropriate settings for the tool segments being used.

Segment A segment is a bending tool that is used as a bending station or part of a bending station. types

A station can consist of multiple segments. Segments can have different shapes or properties.

Fig.: Segment types on a station plan

A Normal segment
B Segment with left/right horn
C Station A station consists of one or more segments of a top tool and a bottom tool. with empty segment

Create a tool setup plan

The two arrow buttons can be used to switch between multiple tool setup plans. If there is only one tool setup plan, these buttons are disabled.

Add

This button adds a new station plan to the tool setup plan.

Using different tools on one tool setup plan

Attention! When using different tools on one tool setup plan there is a risk of collision!

Different tools can be used on the same tool setup plan under the following conditions:

  • They must be of the same height.
  • They must be of the same type (e.g. bending tool or hemming tool).
Delete

This button removes the current tool setup plan. If there is only one tool setup plan, all stations are removed from this plan.

Center

This button centers the whole length of all stations on the tool setup plan.

Attention! The dependent axes are disregarded (backgauges, bending aids, angle measurement system). They must be positioned on the stations later on the appropriate tabs.

Editing a station

The two arrow buttons can be used to switch between multiple stations. If no station is highlighted, the buttons are disabled.

Add station

This button adds a new station to the tool setup plan. If the tool setup plan is empty, the new station is added at the left-hand end of the tool setup plan.

Insertion positions

  • If no station is selected, the new station is added to the right of the last station on the right, at the preset station separation.
  • If a station is selected, the new station is inserted to the right of the selected station, at the preset station separation. If the gap to the next station is not large enough to fit the new station, no station is inserted.
Delete station

This button removes the selected station from the tool setup plan.

Station

Use this button to move the top and bottom tools of a station separately or together. Tap and drag the appropriate tools or station to move them.

Editing the top and bottom tools in each station

In the tab for the top and bottom tools you can make manual changes to the position, length and segmentation of the selected station. The positions of the top tool and bottom tools can be defined jointly or separately.

  • Left-hand / Right-hand position

    • This option is used to define the position of the station on the station plan. The distance dimension entered refers to the left-hand or right-hand side of the station.

    Info: On Hämmerle bending machines, the segmentation of the tool holder means that the position can only be set in steps of 50 mm.

  • Length

    • This option shows the total length of the station. The length is calculated from the sum of the individual segments.

    This button is used to change the length of the station, with segmentation performed automatically at the same time. Use this function as a simple way of adjusting the length of an added station.

    NOTE: For lengths that cannot be achieved precisely with segments, the length of the station is extended to the next larger size based on the available tool segments.

  • Bend The bend is an element of the profile programming and designates a change in shape of the workpiece. A number of attributes can be assigned to a bend: side length, bend angle, bend radius (optional), initial-bend angle (optional), additional attributes (optional). A bend can consist of several bending processes. length

    • Shows the bend length of the side.

  • Tool

    • Provided that more than one tool can be selected, this option lets you change the tool for the bending process.

  • Rotated

    • This option defines the bending tool as rotated. This option is only available if the top or bottom tool has been rotated for one or more bending processes on the Bending sequence The bending sequence refers to the specific order in which all the bending edges on a part are implemented. A bending part can usually be bent using a number of different bending sequences. When the bending sequence is set automatically, the software works out the optimum bending sequence and then assigns it to the bending part. tab.

    If a bending station has this option assigned, those bending processes that are bent on this station with a rotated tool must be assigned using the Assign process function.

    Info: If the tools are rotated for all the bending processes, this option is displayed as selected for all processes, but is disabled.

  • Synchronous

    • If this option is enabled, any change made to the position of a top tool or bottom tool is applied synchronously to the other tool in each case.

  • Horn A horn is a top tool whose tool holder on the left or right side is shorter than the length of the tool tip that is used for bending. tool

    • Defines the strategy for using horn tools.

    • None

      • No horn is used.

    • Left

      • A horn tool is used on the left side of the bending station.

    • Right

      • A horn tool is used on the right side of the bending station.

    • Both sides

      • A horn tool is used on both sides of the bending station.

  • Calculation

    • Defines the segmentation strategy.

    • Do not exceed length

      • The maximum length of the bending station is as long as the current bending process.

      Info: Bend length is displayed in parentheses.

    • Minimize number of segments

      • The bending station is created with as few segments as possible.

Manually setting the position and length of a station
  1. Select the station to edit in Tool setup plan view.
  2. Enable the Synchronous option to apply the changes automatically to the bottom station as well.
  3. Use the Left-hand position option to move the left-hand side of the station to the position you require on the station plan.
  • This button adds the segment selected in the segment display to the station.
  • This button lets you edit the length and type of segments in Edit dialog.

NOTE: On Hämmerle bending machines, tools with a segment length of 55–95mm or horn tools can only be used at the beginning or end of a bending station. If you attempt to set one of these segments in the inside area of a bending station, the following error message appears: Interleaving of segments invalid..

Automatically adjusting the length of a station
  1. Select the station whose length you wish to change in Tool setup plan view.
  2. Select the top tool tab.
  3. Enter the new length of the top tool in the Length box.
  4. Press the button. The top station is adjusted to the specified size.
  5. Repeat steps 2 to 4 for the bottom tool.
