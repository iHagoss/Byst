Technology

This function is used to set the entire bending technology and automatically assign it to the bending part.

Automatically assign bending technology

This function is used to assign the entire bending technology to the bending part. The fully automatic procedure for creating the bending technology involves the following process steps:

  1. Specify bending tools.
  2. Create the bending sequence.
  3. Create a tool setup plan.
  4. Calculate backgauges.
  5. Configure bending aids (only if fitted).
  6. Configure angle measurement system (only if fitted).
Complete technology

This function automatically recalculates all technology data not yet calculated. The process data is checked automatically.

Check technology

This function is used as follows to check the entire bending technology for the part.

Symbol Description
Skip valid sub-areas.
Recalculate automatically any sub-areas that are not valid.
Check automatically any sub-areas that need checking
