Edit

In this dialog box you can set or change the position of the angle measurement system for the selected bending process. Invalid settings are identified by an error symbol .

Function

The functions of the angle measurement system are configured in this area.

Mode

This option enables or disables the angle measurement system based on certain criteria.

  • On

    • Switches the angle measurement system on for all bending processes.

  • First On

    • Switches angle measurement on for the first bending process with identical bend angles.

  • Off

    • Switches the angle measurement system off. Now only the axes of the angle measurement system can be configured.

  • Adopt

    • Adopts the measured values of the bending process configured in the Adopt from process box.

Adopt from process

This option adopts the measured values from the selected bending process.

A bending part with more than one bending angle with the same value is required to adopt the measurement from the LAMS See Angle measurement system system. Activate this feature for the first bending process with the desired angle. You can use the measured values in subsequent bending processes with the same angle value.

Sensor

Use the selected sensors for measuring angles. Be default, both sensors should be used as this ensures greater accuracy.

Every n'th bending part

For a series of identical bending parts, you can enter the number of produced parts after which you want the angle measurement system to check another part.

Axes

This area is used for positioning the angle measurement system along the tool holder.

Use

This option is used to position the angle measurement system or to move it into the parked position. If the Parked option is enabled, all other options for the angle measurement system are disabled.

LZB/F position

This option can be used to position the angle measurement system on the bending station of the selected bending process.

  • B = back
  • F = front
Move

Use this option to tap and drag the angle measurement system to the desired position in the 3D view (See «Using the touchscreen»).

Spring-back compensation

This section is used to configure the correction for material spring-back during bending. It is only enabled if the mode is set to On in the function.

Mode

The spring-back measurement is needed to correct for the spring-back of the material during bending.

  • Measurement On

    • The spring-back angle is measured automatically.

  • Database

    • The spring-back angle from the database is used.

  • Manual

    • The spring-back angle must be entered manually via the Angle option.

Expected angle

Defines the expected angle when measuring spring-back compensation. If the angle is no longer in the defined range, an error message is shown.

  • Database

    • The expected angle is read from the database.

  • Manual

    • You can enter the expected angle in the box below.

Angle

If the Manual attribute is selected in the Expected angle option, the spring-back compensation angle must be entered in this box.
