Auxiliary bend

This dialog is used to add an auxiliary bend to the highlighted bend line. The auxiliary bend can only be used with 2D profiles. An auxiliary bend consists of two bending steps.

  1. The first step is to bend the auxiliary bend with the defined bend angle.
  2. The second step involves bending the auxiliary bend back by the defined bend angle at a later time.

Tip: This function can be used, for example, on U profiles with long sides to prevent the sides from colliding with the tool or the bending machine.

Creating an auxiliary bend
  1. Highlight the bend in the 2D view to which you wish to add an auxiliary bend.
  2. Use the Offset on blank part option to define the distance of the auxiliary bend to the bend line.
  3. Press the Insert button.
  4. Use the Check function to check the bending sequence for collisions.

Flange length

This option defines the distance to the bend line without bend allowance.

Offset on blank part

This option defines the distance to the bend line with half bend allowance.

You can define the offset by entering a number or by moving the slide control.

Buttons
