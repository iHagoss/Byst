Settings

The technology settings for automatic assignment of tools are defined in this dialog.

The settings are applied after running the Automatic or Complete function on the part.

Mode
  • Standard tools

    • This option sets the standard tools for all bending processes.

  • Radius

    • For each bending process, this option uses the tool pair that creates a radius that matches the programmed radius as nearly as possible.

  • Bend allowance The reduction in the side length for bends made on the bending part is called a bend allowance. Bending causes the part to be stretched in a longitudinal direction. Therefore the length of the side of a bend must be reduced again by this amount in the cutting part. The bend allowance, i.e. shortening of the side length, compensates for this stretching.

    • For each bending process, this option uses the tool pair that creates a bend allowance that matches the programmed bend allowance as nearly as possible.

Tolerance

This option defines the tolerance for Radius and Bend The bend is an element of the profile programming and designates a change in shape of the workpiece. A number of attributes can be assigned to a bend: side length, bend angle, bend radius (optional), initial-bend angle (optional), additional attributes (optional). A bend can consist of several bending processes. allowance mode in which the same tools can be used.

If there different bend allowances or radii are within this tolerance, the same tools are used for the bends.

Top/bottom tool

You can change the standard tools for the part in the selection boxes. Details of the selected tool are shown below the two selection boxes.

Info: The bending tools can only be changed if Standard tools mode is enabled.
