Bending sequence

This tab contains functions that can be used to assign the bending sequence to the bending part automatically or manually. A check function lets you check all bending sequences for collisions.

The view is divided into a 3D and a 2D view. You can use various bars in the view to edit the bending part.

2D view

The 2D view displays the developed view of the bending part including all bend lines. The bend lines are shown as dashed lines. If a bending sequence has been assigned to the bend line, it is displayed by the appropriate number. The active bend line is highlighted green. If the Set function is selected, the relevant status indicator is displayed for each bend line (See Â« StatusÂ»).

Tip: Click on a bend line in the 2D view to jump to the appropriate bending process and display it in the 3D view.
