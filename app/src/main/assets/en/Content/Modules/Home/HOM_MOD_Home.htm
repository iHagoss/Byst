Home

The HOME module is displayed immediately after logging into the program. The program always displays the bending part that was last loaded as a 3D view.

The header is displayed in all views. It contains a text box displaying the name of the selected view. You can also use buttons in the header to select the main modules directly (See « Header»).

Properties

This section displays all of the main information about the bending part.

  • You can use the System of units option to adjust the system of units used.
  • The Quantity option defines the number of bending parts to produce.
  • Can be produced indicates whether the bending part can be produced.

    • If the bending part is producible, the LED shows a green light.

    If a bending part cannot be produced, the LED shows a red light. If this is the case, you must switch to the Technology module to recalculate the technology.

  • You can enter additional information about the bending part in the Info 1-3 boxes.

Info: Press the Save button to save changes to the properties. Press the Save as... button to save the bending part under a different name.

Bending time

This section displays all of the information about bending time for the bending part. If there are several bending time tables, you can select them in the Table selection box. The tables are defined in Administrator.

The Weight and the Area of the bending part are also specified. To calculate weight, it is necessary to define the specific weight of the material in Administrator.

Tools

This section displays the top and bottom tools used.

Conflicts

This section displays all errors and warnings regarding the bending technology.

Info: This section is only displayed if there are any conflicts.

Notices

In this section you can enter additional information that is useful for making the bending part.

If the bending part was created in BySoft 7, the section also shows the origin of the bending part and the bending machine with which the bending part is to be produced.
