Bending aids

This tab is used to correct the position of the bending aids.

HB / HF

In this section you can move the position of the rear bending aid (HB) and/or the front bending aids (HF) up or down.

  • HB / HF absolute

    • This option dialog is used to correct the setting for the distance from the top edge of the bending-aid table to the top edge of the bottom tool. The adjustment can be made separately for the left-hand (HB1 or HF1) axis and right-hand (HB2 or HF2) axis, or jointly for both axes as a group.

  • HB / HF relative

    • Fine adjustment of the values entered for the axis positions.

WB

In this dialog box you can move the rear bending aids horizontally to the right or left.

  • WB absolute

    • In this dialog you can adjust the setting for the distance from the bending-aid table to the left-hand outer edge of the press table. The adjustment can be made for the left-hand (WB1) axis and right-hand (WB2) axis.

  • WB relative

    • Fine adjustment of the value entered for the axis position.
