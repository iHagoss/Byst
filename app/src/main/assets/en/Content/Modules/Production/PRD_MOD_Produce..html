Produce - Automatic

This module is used for production of bending parts on the machine. The data automatically imported from the selected bending part is converted and then used for producing the bending parts on the machine. If you wish you can edit the imported data.

The first process step for the selected bending part is displayed in the view window in isometric view. If the machine is initialized correctly and the part can be produced, a green bar appears along the top and bottom edge of the view window. The symbol for operating the pedal is displayed at the bottom right.

Information in the text area of the view window:

  • Product name
  • Quantity
  • Remaining counter (active if target quantity is defined)
  • Remaining time (active as soon as the first part has been produced)

Automatic and single step operating mode

Button to select automatic or single step operating mode.

While machining the processes, you can tap this button to switch between Automatic and Single step operating mode.

Info: The press brake always starts up in automatic mode.

  • Automatic

    • If the middle button is not highlighted, the machine is in automatic mode. The text area in the header displays the entry Produce-Automatic.

    The software starts with the current process and works through all the subsequent processes. The procedure is repeated until the programmed target quantity is reached. If a quantity has not been programmed, the software continues until you cancel the procedure.

  • Single step

    • If the middle button is highlighted green, the machine is in single step mode. The text area in the header displays the entry Produce-Single step.

    The same bending process is always carried out until you select the next bending process. You can navigate through the processes and work through them sequentially.

    In this mode, the parts counter is not enabled even if a quantity has been programmed. Pressing the button again switches the software back to automatic mode.
