Ram

Use this tab to correct the ram. Angle corrections are divided into the following sections:

Y-ram

In this section you can change the position and the parameters of the Y-axis ram (upper beam).

  • Y absolute

    • Use this option to correct the actual position of the ram. This is the distance from the bottom edge of the top tool to the top edge of the bottom tool. The correction can be made separately for the left-hand (Y1) axis and right-hand (Y2) axis, or jointly for both axes as a group.

  • Y relative

    • Fine adjustment of the value entered for the axis position.

M ram (3-point only)

In this section you can change the position of the M-axis ram (bottom tool).

  • M absolute

    • Use this option to correct the actual position of the top edge of the V-shaped system. This is the distance from the top edge of the V-shaped system to the top edge of the press table.

  • M relative

    • Fine adjustment of the value entered for the axis position.

Parameters

Use this section to correct the general ram parameters.

  • Upper dead center

    • Defines the position of the ram at the end of the bending cycle.

    • min: Sheet thickness + 1.0 mm
    • max.: < maximum position Y1 or Y2
  • Switchover point Down

    • Controls the switchover from rapid-speed to slow-speed movement of the upper beam. This option sets the rapid-to-slow switchover at a higher point.

    • min: safety point
    • max.: < maximum position Y1 or Y2
  • Pinching point

    • Defines the position in which the bending part is pinched. This is important when a retraction is programmed or bending aids are enabled.

    • min: 50% sheet thickness
    • max.: < changeover point
  • Switchover point Up

    • Defines how far the upper beam travels upwards at a defined speed before it is switched into rapid travel.

    If the Bend The bend is an element of the profile programming and designates a change in shape of the workpiece. A number of attributes can be assigned to a bend: side length, bend angle, bend radius (optional), initial-bend angle (optional), additional attributes (optional). A bend can consist of several bending processes. speed Up parameter is set to a value <10 mm/s, the Switchover point Up value is set automatically to sheet thickness.

    • min: programmed Y position
    • max: maximum position
  • Bending speed down

    • Defines the bending speed for downward travel

    When a bending aid is enabled, the maximum speed is reduced (reduction depends on the Alpha axis).

    • min: 0.5 mm/s
    • max: 10.0 mm/s
  • Bending speed Up

    • Defines bending speed for upward travel.

    This speed only has any effect when the parameter Switchover point Up is > BDC. If the Bend up parameter is set to a value <10 mm/s, the Switchover point Up value is set automatically to sheet thickness.

    • min: 0.5 mm/s
    • max: 10.0 mm/s
  • Pressing force

    • Defines the force to apply to bend the part.

    • min: 0.0 t
    • max.: < limited by maximum machine pressing force and maximum tool load.
  • Dwell time

    • Defines how long the upper beam remains at bottom dead center (BDC)

    min: 0.0 s

    max: 10 s (configurable)

  • Bottom dead center Stop

    • If this parameter is set to On, the upper beam remains in the bottom dead center until the operator operates the control (pedal or two-handed pushbutton). As soon as the operator releases the control, the upper beam moves to the programmed top dead center.

  • Crowning A system for compensating for deviations that can be traced to deflection of the top and bottom tool during the bending process. factor

    • Adjustment of the degree of crowning

    <1 - reduces crowning

    >1 - increases crowning

    • min: 0.0
    • maximum: < 10
