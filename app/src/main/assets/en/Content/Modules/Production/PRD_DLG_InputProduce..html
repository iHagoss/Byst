Input dialog box

This dialog is used to correct values. Values are only adopted if they lie within the range of the preset minimum and maximum values.

  • The Minimum and Maximum buttons set the value to the minimum or maximum permissible value.
  • The Default button resets the value to the default value.
  • The Õ button deletes the last character of the value.
  • The C button deletes the whole value.
  • The a b/c button can be used to enter angle corrections in degrees and minutes, for example.

    • Example:

    enter 90° 15' as follows.

    1. Input: 90
    2. Button: a b/c
    3. Input: 15
    4. Button: a b/c
    5. Input 60
  • The ± button inverts the sign of the value.
  • The A buttons can be used to assign the correction to the respective function. The window then closes automatically and the new values are active.

    • Info: The buttons are only displayed in the dialog for the relevant functions.

    • Process

      • Entered value is adopted for the current process.

    • Parallel

      • Entered value is adopted for all bends that have the same angle on the same section

    • Station A station consists of one or more segments of a top tool and a bottom tool.

      • Entered value is adopted for all processes in the current station

    • Part

      • Entered value is adopted for all processes of the workpiece
