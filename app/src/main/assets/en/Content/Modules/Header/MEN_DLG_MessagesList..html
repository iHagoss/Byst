Message list

WARNING: Work on electrical equipment on the machine must only be performed electrical experts in accordance with the electrotechnical rules.

All pending messages are displayed as a table in the message list. The icon preceding the message indicates the category and the from which the message originates.

Symbol Category
There are no pending messages.
There is a pending status message. Status messages inform the user about the current process on the machine.

There is a pending warning message. Warnings indicate faults or system errors, the system can be used with certain limitations.
There is a pending error message. Error messages indicate faults or system errors. The system is not ready for use.

Functions in the message list

Symbol Meaning
Clears the pending messages once the cause has been rectified.
