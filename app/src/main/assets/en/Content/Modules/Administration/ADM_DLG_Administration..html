Administration - Favorites

This dialog can be used to select parameters and add them to a list of favorites.

The favorites are displayed in Production view in the Correct -> Favorites dialog.

Procedure:

  1. Select the appropriate parameter in the Available parameters section.
  2. Press the Add button to add the parameter to the Selected parameters section.
  3. Repeat steps 2â€“3 until you have added all necessary parameters as favorites.

Buttons

  • Moves the selected parameter down one line.
  • Moves the selected parameter up one line.

Tip: You can define parameters as favorites that you need to edit or change most often during the production process.
