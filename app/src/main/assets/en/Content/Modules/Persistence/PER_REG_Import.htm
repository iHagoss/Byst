Import bending part

This tab is used to import bending parts into the Home menu. You can enter the name of the part in the text box to filter the display.

File type
  • All known file formats

    • You can use this function to import from a database a bending part that was created in the .bpx or .bendpart format.

  • BPX files

    • You can use this function to import from a database a bending part that was created using MMC ByBend or BySoft 6.x.

  • Bendpart files

    • This function imports from a database a bending part that was created with the BySoft 7 CAD/CAM program.

  • DXF files

    • This function imports a bending part in DXF format with bending information from a database.

Sort by
  • Name

    • Sorts the bending parts by part name.

  • Date

    • Sorts the bending parts by creation date.

  • Size

    • Sorts the bending parts by file size.

  • Ascending

    • Starting from the beginning of the alphabet.

  • Ascending

    • Starting from the end of the alphabet.

Conflicts during import

Any name conflicts that arise during the automatic import of *.bendpart files are displayed.

The following solutions are available:

  • Delete the original file. The file is not imported.
  • Import the file and overwrite the existing bending part with the same name.

    • This automatically unloads and deletes the part that is already loaded. Then the new part is imported and loaded automatically under the same name.

    Attention! Any changes to technology, corrections, etc., made to the loaded part are lost in the process.

Set favorites

Saves the selected path as a favorite. This path is always displayed as the default path on this tab.

Tip: Favorites make it easier to find bending part to import that are stored in a default path.
