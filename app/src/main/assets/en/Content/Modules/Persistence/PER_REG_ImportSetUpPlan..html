Import

This dialog is used to import the tool setup plan for an existing bending part into the current bending part.

The main view displays all bending parts in the pool of parts as 2D images. You can enter the name of the part in the text box to filter the display.

Info: The filter is applied even as you type.

Procedure:

  1. Select the bending part whose tool setup plan you want to import.
  2. Press the Import button. The program imports the tool setup plan and assigns it to the bending part as a new tool setup plan.

Info: Any existing tool setup plans are retained and must be deleted manually.

Sort by

The listed bending parts can be sorted according to the following headings:

  • Name

    • Shows the name of the part.

  • Description

    • You can enter a more detailed description of the current object in this field.

  • Created

    • Displays the date of the last update.

  • Info 1-3

    • This field may contain extra information about the current object.

  • Ascending

    • Starting from the beginning of the alphabet.

  • Descending

    • Starting from the end of the alphabet.

Import

Imports to the tool setup plan of the selected bending part into the current bending part.

Search filter

In this dialog box you can configure filter settings for the input dialog.

  • Search in

    • The search includes all highlighted options when you enter text in the input box.

  • Search operator

    • Defines how filter options are combined.

  • Created between

    • Includes all bending parts created between the two dates.

  • Changed between

    • Includes all bending parts edited between the two dates.

Back

Closes the dialog and returns to Tool setup plan view. No tool setup plan is imported.
