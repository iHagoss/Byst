Program login

The main screen with the Login dialog box is displayed when you launch the software. To log into the program, you must select a user from the user carousel. Additionally, you can enter a four-digit PIN code.

User login

  1. Select a user or enter a PIN code.
  2. Press the Login button. The program switches to Job - Prepare view

Info: Once you have logged on you can access those functions available for your user permissions.

NOTE: The operating instructions for the machine describe which activities and views can be accessed for which user permissions. The user permissions can be changed in the Administrator software module.

Changing the PIN
  • Press the Change PIN button.
  • Enter old PIN code and press OK to confirm.
  • Enter new PIN code and press OK to confirm.
  • Re-enter new PIN code and press OK to confirm.

If all details are entered correctly, you can now log into the program with the new PIN code.
