Tool setup plan

This view presents information about the configured stations.

The lower area of the screen displays the individual stations. As soon as you tap the top tools or bottom tools in a station view, they are highlighted in green. The selected tools are shown graphically in cross-section together with their relevant details.

The information displayed in information lines Info 1-3 has been entered previously in the relevant area of the Administrator for the selected tool.

  • The Park button opens the dialog box used to move components to the parked position (See «Park»).
  • The Bend line A bend line refers to a line along which the side is bent. laser button opens the dialog used to adjust the laser beam (See «Bend line laser»).
