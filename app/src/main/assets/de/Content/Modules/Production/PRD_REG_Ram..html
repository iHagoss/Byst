Stössel

In diesem Register können Korrekturen des Stössels vorgenommen werden. Die Winkelkorrekturen sind in folgende Bereiche unterteilt:

Y-Stössel

In diesem Bereich kann die Position und die Parameter des Stössels der Y-Achse (Oberwange) korrigieren werden.

  • Y-Absolut

    • Mit dieser Option kann die Ist-Wert Position des Stössels korrigiert werden. Dies ist der Abstand der Unterkante vom Oberwerkzeug zur Oberkante des Unterwerkzeugs. Die Korrektur kann für die linke (Y1) und rechte (Y2) Achse getrennt oder für beide als Achsgruppe ausgeführt werden.

  • Y-Relativ

    • Feinjustierung der eingetragenen Achsposition.

M-Stössel (nur 3-Punkt)

In diesem Bereich kann die Position des Stössels der M-Achse korrigiert werden (Unterwerkzeug).

  • M-Absolut

    • Mit dieser Option können die Ist-Wert Position der Oberkante des Keilsystems korrigiert werden. Dies ist der Abstand der Oberkante vom Keilsystem zur Oberkante des Pressentischs.

  • M-Relativ

    • Feinjustierung der eingetragenen Achsposition.

Parameter

In diesem Bereich können die allgemeinen Parameter des Stössels korrigiert werden.

  • Oberer Totpunkt

    • Definiert die Position des Stössels am ende des Biegezyklus.

    • min.: Blechdicke + 1.0 mm
    • max.: < Maximalposition Y1 oder Y2
  • Umschaltpunkt ab

    • Beeinflusst die Umsteuerung von der Eilganggeschwindigkeit auf die Kriechgangsgeschwindigkeit der Oberwange. Diese Option setzt die Umsteuerung Eilgang auf Kriechgang höher.

    • min.: Sicherheitspunkt
    • max.: < Maximalposition Y1 oder Y2
  • Klemmpunkt

    • Definiert die Position, um das Biegeteil zu klemmen. Dies ist wichtig, wenn ein Rückzug Als Rückzug wird die Rückwärtsbewegung des Anschlagarms bezeichnet, um eine Kollision des Anschlagfingers mit dem Biegeteil zu verhindern. Die Rückzugbewegung wird ausgeführt, nachdem das Biegeteil zwischen Ober- und Unterwerkzeug geklemmt wurde. programmiert oder Biegehilfen aktiviert sind.

    • min.: 50% Blechdicke
    • max.: < Umschaltpunkt
  • Umschaltpunkt auf

    • Definiert, wie weit die Oberwange mit einer definierten Geschwindigkeit nach oben fährt, bevor in Eilgang umgeschaltet wird.

    Wird der Parameter Biegegeschwindigkeit auf auf einen Wert <10 mm/s gesetzt, wird der Wert Umschaltpunkt auf automatisch auf die Blechdicke gesetzt.

    • min.: Programmierte Y-Position
    • max.: Maximalposition
  • Biegegeschwindigkeit ab

    • Definiert die Biegegeschwindigkeit abwärts.

    Bei aktivierter Biegehilfe reduziert sich die maximale Geschwindigkeit (abhängig von der Alpha-Achse).

    • min.: 0.5 mm/s
    • max.: 10.0 mm/s
  • Biegeschwindigkeit auf

    • Definiert die Biegegeschwindigkeit aufwärts.

    Diese Geschwindigkeit wirkt nur, wenn auch der Parameter Umschaltpunkt auf > UT ist. Wird der Parameter Biegen auf auf einen Wert <10 mm/s gesetzt, wird der Wert Umschaltpunkt auf automatisch auf die Blechdicke gesetzt.

    • min.: 0.5 mm/s
    • max.: 10.0 mm/
  • Presskraft

    • Definiert die Kraft die aufzuwenden ist, um das Biegeteil zu biegen.

    • min.: 0.0 t
    • max.: < Begrenzt durch die maximale Presskraft der Maschine und die maximale Belastung der Werkzeuge.
  • Presszeit

    • Definiert, wie lange die Oberwange im unteren Totpunkt (UT) stehen bleibt.

    min.: 0.0 s

    max.: 10 s (konfigurierbar)

  • UT Halt

    • Ist der Parameter auf Ein gesetzt, bleibt die Oberwange im unteren Totpunkt stehen, solange der Bediener das Bediengerät betätigt (Fußschalter oder Zweihand-Taster). Sobald das Bediengerät losgelassen wird, fährt die Oberwange zum programmierten oberen Totpunkt.

  • Kein Stösselhub

    • Ist der Parameter aktiviert, wird während des aktuellen Prozesses die Oberwange nicht bewegt. Dies erlaubt auf Applikationsebene komplexe Bewegungen der Peripherie in Vorbereitung für den nächsten Prozessschritt. Nützlich kann dies beispielsweise bei Roboterbiegezellen zur Umpositionierung von Achsen sein.

  • Bombierungsfaktor

    • Anpassung der Bombierung Ist ein System für den Ausgleich von Abweichungen, die auf ein Durchbiegen von Ober-und Unterbalken während des Biegeprozesses zurückzuführen sind.

    <1 - Bombierung wird kleiner

    >1 - Bombierung wird grösser

    • min.: 0.0
    • max.: < 10
