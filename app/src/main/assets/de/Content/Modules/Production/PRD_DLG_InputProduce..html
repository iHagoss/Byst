Eingabedialog

In diesem Dialog können Wertkorrekturen vorgenommen werden. Es werden nur die Werte übernommen, die im Bereich der vorgegebenen Minimum- und Maximumwerten liegen.

  • Die Schaltflächen Minimum und Maximum setzen den Wert auf den minimalen oder maximal zulässigen Wert.
  • Die Schaltfläche Standard setzt den eingegebenen Wert wieder auf den Standardwert zurück.
  • Die Schaltfläche Õ löscht das letzte Zeichen des angegebenen Werts.
  • Die Schaltfläche C löscht den gesamten Wert.
  • Mit der Schaltfläche a b/c können beispielsweise Winkelkorrekturen in Grad und Minuten angeben werden.

    • Beispiel:

    90° 15' wird wie folgt eingegeben.

    1. Eingabe: 90
    2. Taste: a b/c
    3. Eingabe: 15
    4. Taste: a b/c
    5. Eingabe 60
  • Die Schaltfläche ± kehrt das Vorzeichen des Werts um.
  • Mit den Schaltflächen A, kann die Korrektur der jeweiligen Funktion zugewiesen werden. Das Fenster schliesst danach automatisch und die neuen Werte sind aktiv.

    • Info: Die Schaltflächen werden nur bei relevanten Funktionen im Dialog angezeigt.

    • Prozess

      • Eingabe wird für den aktuellen Prozess übernommen.

    • Parallel

      • Eingabe wird für alle Biegungen mit gleichem Winkel auf dem gleichen Schnitt übernommen

    • Station Eine Station besteht aus ein- oder mehreren Segmenten eines Ober- und Unterwerkzeugs.

      • Eingabe wird auf alle Prozesse der aktuellen Station übernommen

    • Teil

      • Eingabe wird auf alle Prozesse des Werkstücks übernommen
