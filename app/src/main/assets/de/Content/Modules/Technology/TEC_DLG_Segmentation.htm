Segment bearbeiten

In diesem Dialog können alle verfügbaren Werkzeugsegmente einer Station Eine Station besteht aus ein- oder mehreren Segmenten eines Ober- und Unterwerkzeugs. hinzugefügt werden.

  • Segment Ein Segment ist ein Biegewerkzeug, dass als Biegestation oder Teil einer Biegestation verwendet wird.

    • Mit dieser Option wird das gewählte Segment der Biegestation Eine Biegestation besteht immer aus einem oberen und unteren Biegewerkzeug und ist Bestandteil eines Stationsplans. hinzugefügt.

  • Leersegment Ein Leersegment ist ein definierter freier Bereich innerhalb einer Biegestation. Leersegmente werden für verbundene Biegelinien zwischen zwei Schenkeln verwendet, wenn sich zwischen den Schenkeln eine Kontur befindet, welche nicht gebogen werden darf. Ein Leersegment muss immer von zwei normalen Segmenten umgeben sein.

    • Mit dieser Option wird ein Leersegment beliebiger Länge der Station hinzugefügt.
