Hilfsbiegung

In diesem Dialog wird zur markierte Biegelinie Als Biegelinie wird eine Linie bezeichnet an der entlang die Biegung der Schenkel erfolgt. eine Hilfsbiegung Die Hilfsbiegung wird bei bestimmten Biegeformen vorläufig eingefügt und dann wieder zurückgebogen, um die Endform eines Biegeteils zu ermöglichen. eingefügt. Die Hilfsbiegung kann nur bei 2D-Profilen eingesetzt werden. Eine Hilfsbiegung besteht aus zwei Biegeschritten.

  1. Im ersten Schritt wird die Hilfsbiegung mit dem angegebenen Biegewinkel gebogen.
  2. Im zweiten Schritt wird die Hilfsbiegung zu einem späteren Zeitpunkt wieder um den angegebenen Biegewinkel zurückgebogen.

Tip: Diese Funktion kann beispielsweise bei U-Profilen mit langen Schenkeln verwendet werden, um Kollisionen der Schenkel mit dem Werkzeug oder der Biegemaschine zu vermeiden.

Winkel

Mit dieser Option wird der Biegewinkel (α) der Hilfsbiegung bestimmt. Zu einem späteren Zeitpunkt wird die Biegung In einem zweiten Schritt um diesen Winkel wieder zurückgebogen.

Schenkellänge

Mit dieser Option wird der Abstand zur Biegelinie ohne Verkürzung Als Verkürzung wird die Reduzierung des Schenkelmasses der Biegungen am Biegeteil bezeichnet. Durch das Biegen wird das Teil in der Länge gestreckt. Deshalb muss der Schenkel einer Biegung um dieses Mass beim Schneidteil wieder verkürzt werden. Durch das Verkürzen wird diese Streckung wieder kompensiert. angegeben.

Versatz auf Zuschnitt

Mit dieser Option wird der Abstand zur Biegelinie mit halber Verkürzung angegeben.

Der Versatz kann numerisch oder durch Verschieben des Schiebereglers angegeben werden.

Schaltflächen
  • Löschen

    • Mit dieser Funktion wird die gewählte Hilfsbiegung entfernt.

  • Invertieren

    • Mit dieser Funktion wird der Winkel der Hilfsbiegung umgekehrt. Das Biegeteil muss danach vertikal um 180° gedreht auf das Unterwerkzeug gelegt werden.

  • Einfügen

    • Mit dieser Funktion wird eine Hilfsbiegung mit zwei Biegungen anhand der eingestellten Optionen erzeugt.
