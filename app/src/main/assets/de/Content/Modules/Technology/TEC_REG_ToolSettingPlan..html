Einrichtplan

In diesem Register wird der Einrichtplan Der Einrichtplan besteht aus mindestens einem Stationsplan. Jeder Stationsplan besteht wiederum aus mindestens einer Biegestation auf dem das Biegeteil gebogen wird. Auf einem Einrichtplan sind alle Werkzeuge für den gesamten Biegeprozess definiert. für das Biegeteil erstellt. Ein Einrichtplan kann aus ein- oder mehreren Stationsplänen bestehen. Durch eine Prüffunktion können alle erstellten Stationspläne auf Gültigkeit geprüft werden.

Die Ansicht ist in ein 3D- und 2D-Ansicht unterteilt. Das Biegeteil kann über verschieden Leisten in der Ansicht verändert werden.
