Setzen

In diesem Dialog kann die Biegefolge Als Biegefolge wird die Abarbeitung aller Biegekanten eines Teils in einer bestimmten Reihenfolge bezeichnet. Ein Biegeteil kann meist durch mehrere Biegefolgen gebogen werden. Beim automatischen Setzen der Biegefolge ermittelt das Programm die optimalste Biegefolge und weist es dem Biegeteil zu. manuell auf ein Teil gesetzt werden. Das Setzen der Biegefolge kann auf zwei Arten geschehen.

  • Durch die Schaltflächen am unteren Rand des Dialogs.
  • Durch Tippen auf eine beliebige Biegelinie Als Biegelinie wird eine Linie bezeichnet an der entlang die Biegung der Schenkel erfolgt. in der 2D-Ansicht unterhalb vom Ansichtsfenster, wird die Biegelinie als nächste Biegung Die Biegung ist ein Element der Profilprogrammierung und bezeichnet eine Formänderung im Werkstück. Einer Biegung können mehrere Attribute zugeordnet werden: Schenkellänge, Biegewinkel, Biegeradius (optional), Anbiegewinkel (optional), zusätzliche Attribute (optional). Eine Biegung kann aus mehreren Biegeprozessen bestehen. in der Biegefolge gesetzt und grün markiert. Der Status der aktiven Biegelinie wird in der Tabelle und in der 2D-Ansicht mit einem Symbol markiert.

Attention! Biegelinien ohne Statussymbol können in der Regel der Biegefolge nicht ohne eine Kollision hinzugefügt werden.

Info:

Die jeweils gewählte Biegung kann im Ansichtsfenster in einer 3D-Vorschau angezeigt werden. Damit die Biegung als Vorschau angezeigt wird, muss die Schaltfläche Vor Ausführung in der Ansichtsleiste deaktiviert sein.

Biegefolge in der 2D-Ansicht setzen

Durch Klicken auf eine Biegelinie wird diese der Biegefolge hinzugefügt. Das Hinzufügen geschieht in der Reihenfolge, wie auf die Biegelinien geklickt wird.

Gesetzte Biegelinien werden grau dargestellt. Die Reihenfolge in der Biegefolge wird durch Nummern mit dem entsprechenden Statussymbol auf der Biegelinie angezeigt (See « Status»).

Zeigt die Reihenfolge der Biegeprozesse mit einer Laufnummer an.

Winkel

Zeigt den Winkel der Biegung an. Der Winkel 0 wird nur bei der Endbiegung einer Flachbiegung Dieser Biegetyp biegt das Blech mit einem Biegewinkel von 180° flach zusammen. Der Biegevorgang wird nacheinander in den zwei Schritten «Vorbiegen» und «Flachbiegen» vollzogen. Diese Schritte können auch getrennt voneinander ausgeführt werden. verwendet.

Biegeoperation

Zeigt die Art der Biegung im Biegeprozess an.

  • Normalbiegung Die Normalbiegung ist die übliche Methode, eine Biegung zu erstellen. Die Biegung wird durch den Biegewinkel und einem optionalen Anbiegewinkel bestimmt.
  • Flachbiegung
  • Vorbiegung Dieser Biegetyp erstellt eine Biegung in zwei Schritten. Die Biegung wird zuerst mit einem Anbiegewinkel gebogen und zu einem späteren Zeitpunkt mit dem Endwinkel fertig gebogen. Verwenden Sie das Vorbiegen bei komplexen Biegeteilen, um Kollisionen während dem Biegeprozess zu vermeiden.
  • Hilfsbiegung Die Hilfsbiegung wird bei bestimmten Biegeformen vorläufig eingefügt und dann wieder zurückgebogen, um die Endform eines Biegeteils zu ermöglichen.
  • Stepradiusbiegung
