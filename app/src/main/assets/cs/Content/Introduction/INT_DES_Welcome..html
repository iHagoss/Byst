BySoft Cell Control Bend

Version: 1.0.0

Vítejte

Vítejte u BySoft Cell Control Bend, obslužného softwaru pro ohýbací stroje Bystronic.

Pomocí BySoft Cell Control Bend můžete efektivně a pohodlně obsluhovat ohýbací stroje. Program je navržen k obsluze přes pracovní plochu na obrazovce obslužného terminálu. Ovládání funkcí programu se tak může provádět přímo na obrazovce.

  • Co je nového: Informujte se o nových funkcích programu v předložené verzi.
  • Struktura pracovní plochy: Seznamte se s uživatelským rozhraním BySoft Cell Control Bend.
  • Používání dotykové obrazovky: Seznamte se s ručním ovládáním programu.
