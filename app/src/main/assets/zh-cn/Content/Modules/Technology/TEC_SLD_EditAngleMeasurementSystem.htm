编辑

在该对话框中，可针对选定的折弯过程 所需的一个或多个折弯的所有行动或程序被称为折弯过程。设置和更改角度测量系统 角度测量系统能够在折弯过程中将依赖于材料的折弯角度的变化最小化。这确保了在零件的整个长度上的优异的角度精度。的位置。通过错误符号 识别无效设置。

功能

在该区域内设置角度测量系统的功能。

模式

该选项根据特定标准启用或禁用角度测量系统。

  • 开

    • 打开角度测量系统以进行所有折弯 折弯是指配置文件编程要素并且指定工件形状的变化。可以被分配给折弯的一系列属性：边长，折弯角度，折弯半径（可选），初始折弯角度（可选），额外的属性（可选）。折弯可包括若干折弯过程。过程。

  • 首次开

    • 用相同的折弯角度开启第一个折弯过程的角度测量。

  • 关

    • 关闭角度测量系统。现在只能配置角度测量系统的轴。

  • 接受

    • 采用从过程采用框中所配置的折弯过程的测量值。

采用过程

该选项采用选定折弯过程中的测量值。

传感器

使用选定的传感器测量角度。默认情况下，应使用两个传感器，因为这可确保更高的准确性。

每 n 个折弯零件

对于一系列相同的折弯零件，可以在此输入想要通过角度测量系统来检查其他零件之后的已生产零件的数量。

轴

该区域用于定位沿着模具架运行的角度测量系统的位置。

使用

该选项用于定位角度测量系统或将其移动至停车位置。如果停止选项已启用，则角度测量系统的全部其他选项均禁用。

LZB/F 位置

该选项可用于在选定的折弯过程的折弯工位 折弯工位总是包括上部折弯模具和下部折弯模具，并且包含在一个工位方案中。上定位角度测量系统。

  • B = 后
  • F = 前
移动

使用此选项在3D视图中轻按角度测量系统并将其拖动到所需位置 (See «使用触摸屏»)。

回弹补偿

该区域用于配置折弯过程中材料回弹的校正。仅当模式在功能中被设定为开时，才可启用。

模式

在折弯期间有必要在校正材料回弹时进行回弹测量。

  • 打开测量系统

    • 自动测量回弹角度。

  • 数据库

    • 数据库中的回弹角度已使用。

  • 手动

    • 回弹角度必须通过角度选项手动输入。

预期角度

定义测量回弹补偿时的预期角度。如果角度不在定义的范围内，则显示出错讯息。

  • 数据库

    • 预期的角度是从数据库中读取的。

  • 手动

    • 您可以在下面的框中输入预期的角度

角度

如果在期望角度选项中选择了手动属性，则必须在该框中输入回弹补偿角度。
