Butées

Cet onglet permet de réaliser des corrections des butées. Les corrections de butée sont réparties sur les zones suivantes :

XB

Cette rubrique permet de corriger la position du doit de butée arrière. Les doigts de butée arrière peuvent être déplacés vers la ligne de pliage vers l'avant ou vers l'arrière.

  • XB absolu

    • Cette option permet de corriger la distance configurée du point de butée sélectionné avec le centre de l'outil inférieur. La correction peut être exécutée pour l'axe gauche (XB1) et l'axe droite (XB2) ou pour les deux axes comme groupe.

  • XB relatif

    • Ajustement fin de la position d'axe entrée.

ZB

Cette rubrique vous permet de déplacer vers la droite ou la gauche la position des butées arrière.

  • ZB absolu

    • Cette option permet de corriger la distance définie du point de référence du doigt de butée correspondant par rapport à l'arête extérieure gauche de la table de presse.

  • ZB relatif

    • Ajustement fin de la position d'axe entrée.

RB

Ce dialogue vous permet de déplacer vers le haut ou le bas la position des butées arrière.

  • RB absolu

    • Cette option permet de modifier la distance configurée du point de butée sélectionné en fonction de l'outil inférieur.

  • RB relatif

    • Ajustement fin de la position d'axe entrée.

Paramètres

Cette brique permet de corriger les paramètres Exécuter le retour, Exécuter le périphérique et Vitesse des butées.

  • Exécuter le retour

    • Le retour entraîne le retrait des axes de butée X. Deux positions sont disponibles.

  • Position de fin de course

      • Le tablier supérieur attend dans le point de serrage jusqu'à ce que les axes X aient atteint la position.

  • Mobile

      • Le tablier supérieur s'arrête brièvement sur le point de serrage, puis les axes X et le tablier supérieur continuent simultanément.

  • Exécuter le périphérique

    • Cette option définit quand les axes du périphérique peuvent démarrer.

  • CAUTION:

    Les axes de périphérique en mouvement peuvent toucher la pièce de pliage en direction de l'opérateur et blesser ce dernier. Lorsque vous corrigez les axes du périphérique, tenez compte de la manipulation de la pièce à plier.

    Pour les pièces à plier avec des côtés longs derrière l'outil de pliage, il convient d'utiliser l'option Retard afin de provoquer le déplacement des axes du périphérique par l'opérateur.

  • Retard

      • Les axes du périphérique ne démarrent qu'après une action de l'opérateur (acceptation par ex. avec la pédale).

  • Point mort haut

      • Lorsque le tablier supérieur atteint le point mort haut, les axes du périphérique avancent jusqu'à la position programmée suivante.

  • Point de blocage

      • Lorsque le tablier supérieur dépasse le point de serrage lors du démarrage, les axes du périphérique avancent déjà jusqu'à la position programmée suivante.

  • Vitesse

    • Cette option détermine la vitesse de déplacement des axes en pour cent.
