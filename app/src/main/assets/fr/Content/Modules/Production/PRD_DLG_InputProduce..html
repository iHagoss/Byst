Dialogue de saisie

Cette boîte de dialogue permet de réaliser des corrections de valeurs. Seules les valeurs comprises entre les minima et les maximas prédéfinis sont validées.

  • Les boutons Minimum et Maximum définissent la valeur minimale ou maximale admissible pour la valeur.
  • Le bouton Standard réinitialise la valeur entrée à la valeur par défaut.
  • Le bouton Õ efface le dernier caractère de la valeur indiquée.
  • Le bouton C efface la valeur complète.
  • Le bouton a b/c permet par exemple d'indiquer des corrections d'angle en degrés et en minutes.

    • Exemple :

    Entrer 90° 15' de la manière suivante.

    1. Saisie : 90
    2. Bouton : a b/c
    3. Saisie : 15
    4. Bouton : a b/c
    5. Saisie 60
  • Le bouton ± inverse le signe de la valeur.
  • Les boutons A permettent de corriger la fonction correspondante. La fenêtre se referme ensuite automatiquement et les nouvelles valeurs sont activées.

    • Info: Les boutons ne sont affichés dans la boîte de dialogue que pour les fonctions pertinentes.

    • Processus

      • La saisie est validée pour le processus actuel.

    • Parallèle

      • La saisie est validée pour tous les pliages avec un angle identique sur la même coupe

    • Station Une station comporte un ou plusieurs segments d'un outil supérieur ou inférieur.

      • L'entrée est validée sur tous les processus de la station actuelle

    • Pièce

      • L'entrée est validée sur tous les processus de la pièce à usiner
