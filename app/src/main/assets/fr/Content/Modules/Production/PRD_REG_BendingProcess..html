Processus de pliage

Cet onglet permet de corriger les paramètres pour le processus de pliage.

Bombage

Le bombage peut être modifié entre la valeur minimale de 0 et la valeur maximale de 10.

  • <1 = Diminution du bombage
  • >1 = Augmentation du bombage
Compteur de cycle

Ce champ d'entrée permet de corriger le nombre de cycles.

  • 0 minimum (processus dépassé)
  • 10 000 = Maximum
Epaisseur de la protection de tôle

Ce champ d'entrée permet de corriger l'épaisseur de la garniture de protection de la tôle.

Pupitre de commande

Ce dialogue permet de choisir entre une ou deux pédales. Confirmez ensuite à l'aide du bouton la fonction pour laquelle l'entrée doit être validée.

Il est possible de sélectionner les fonctions suivantes :

  • Processus

    • La saisie est validée pour le processus actuel.

  • Parallèle

    • La saisie est validée pour tous les pliages avec un angle identique sur la même coupe

  • Station Une station comporte un ou plusieurs segments d'un outil supérieur ou inférieur.

    • L'entrée est validée sur tous les processus de la station actuelle

  • Pièce

    • L'entrée est validée sur tous les processus de la pièce à usiner
