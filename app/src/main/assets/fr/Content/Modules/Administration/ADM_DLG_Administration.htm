Administration - Favoris

Dans cette boîte de dialogue, des paramètres peuvent être sélectionnés et ajoutés sous forme de favoris dans une liste.

Les favoris sont affichés dans la vue Production dans la boîte de dialogue Corriger -> Favoris.

Procédure :

  1. Sélectionner le paramètre souhaité dans la zone Paramètres disponibles.
  2. Le bouton Ajouter permet d'ajouter de paramètre à la zone Paramètres sélectionnés.
  3. Répéter l'étape 2 à 3 jusqu'à ce que tous les paramètres requis aient été ajoutés sous forme de favoris.

Boutons

  • Décale le paramètre sélectionné d'une position vers le bas.
  • Décale le paramètre sélectionné d'une position vers le haut.

Tip: Il est possible de définir comme favoris les paramètres pour lesquels les ajustements ou les modifications les plus fréquents doivent être réalisés pendant le processus de production.
