Configurations

Ce dialogue définit les critères de calcul automatique du système de mesure angulaire.

Fonction

Utilisation

Cette option permet de définir le réglage de base du système de mesure des angles.

  • Positionné - Toutes les mesures désactivées

    • Le système de mesure angulaire est désactivé pour tous les processus de pliage et est positionné sur la station de pliage actuelle.

  • Positionné - Toutes les mesures activées

    • Le système de mesure angulaire est activé pour tous les processus de pliage et est positionné sur la station de pliage actuelle.

  • Positionné - Par angle de pliage première mesure activée

    • Le système de mesure angulaire est activé et est positionné sur la station actuelle. Lorsque plusieurs angles de pliage sont pliés à la suite, seul le premier angle de pliage est mesuré jusqu'à ce qu'un nouvel angle soit plié.

  • Immobilisé

    • Cet attribut vous permet de déplacer le système de mesure des angles sur la position de garage. Le système de mesure des angles reste en position de garage pour le processus actuel jusqu'à ce que l'attribut Immobilisé soit désactivé manuellement à l'aide de la fonction Usinage dans l'option Utilisation ou qu'une autre utilisation ait été sélectionnée.

Info: Le positionnement du système de mesure angulaire est réalisé automatiquement à l'aide d'un servomoteur.

Chaque énième pièce de pliage

Dans une série de pièces de pliage identiques, il est possible d'indiquer ici après combien de pièces produites une pièce supplémentaire doit être contrôlée par le système de mesure angulaire.

Compensation de l'effet de retour

Mode

La mesure de l'effet de retour est requise pour compenser le retour du matériel pendant le pliage.

  • Mesure activée

    • Ceci permet de définir automatiquement l'angle de retour de pliage.

  • Base de données

    • L'angle de retour de pliage est utilisé à partir de la base de données.

  • Manuel

    • L'angle de retour de pliage doit être indiqué manuellement à l'aide de l'option Angle.

Angle attendu

Définit l'angle attendu pour la mesure de compensation de l'effet de retour élastique. Si l'angle n'est plus dans la plage indiquée, un message d'erreur est émis.

  • Base de données

    • L'angle attendu est utilisé à partir de la base de données.

  • Manuel

    • L'angle attendu peut être entré dans le champ situé au-dessous.

Angle

Si vous sélectionnez l'attribut Manuel pour l'optionAngle attendu, vous devez entrer dans ce champ l'angle de l'effet de retour élastique.

Optimisation

Position LZ

On essaie d'utiliser les positions LAMS Voir Système de mesure des angles identiques dans la mesure du possible dans le sens LZ.

Fenêtre de position

Définit la distance maximum de l'axe dans le sens LZ dans lequel le système de mesure des angles peut être déplacé automatiquement.
