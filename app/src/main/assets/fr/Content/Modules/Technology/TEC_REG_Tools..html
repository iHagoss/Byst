Outils

Cet onglet définit les outils de pliage permettant de créer les pliages. L'affectation des outils de pliage peut être réalisée automatiquement ou manuellement.

Le processus de pliage en cours de sélection est affiché dans la vue 2D avec une ligne de pliage verte. Une ou plusieurs opérations de pliage peuvent être affectées à chaque processus de pliage.

Le processus de pliage est sélectionné en cliquant sur une ligne de pliage.

Affichage des outils pour les processus de pliage

L'affectation des outils aux différents processus de pliage est affichée sous la vue 2D. Cliquez sur le processus de pliage pour ouvrir la boîte de dialogue Modifier.

Si des avertissements ou des erreurs sont en cours pour l'outil de pliage utilisé, ceci est indiqué à l'aide d'une icône en regard de l'outil de pliage. Déplacer la souris au-dessus de l'icône d'avertissement pour afficher des informations sur l'avertissement.

Tip: Si l'avertissement ou l'erreur peut être ignorée pour le processus de pliage, ils peuvent être acquittés dans la boîte de dialogue Modifier (See «Boutons»).

NOTE: Un autre outil adapté pour le processus de pliage doit être choisi si le message n'est pas validé.

Opération de pliage

Indique le type de pliage du processus de pliage.

  • Pliage normal Le pliage normal désigne la méthode habituelle pour créer un pliage. Le pliage est défini par l'angle de pliage et par un angle de pliage optionnel.
  • Pliage à plat Ce type de pliage plie à plat la tôle avec un angle de pliage de 180 °. Le pliage est réalisé en deux étapes successives « Prépliage » et « Pliage à plat » Ces étapes peuvent également être exécutées séparément l'une de l'autre.
  • Prépliage Ce type de pliage crée un pliage en deux temps. Le pliage est réalisé tout d'abord avec un angle de pliage, après quoi il est finalisé avec l'angle d'extrémité. Utilisez le prépliage pour les pièces de pliage complexes afin d'éviter les collisions pendant le processus de pliage.
  • Pliage auxiliaire Le pliage auxiliaire est inséré provisoirement sur des formes de pliage définies puis repliées afin de permettre la forme d'extrémité d'une pièce de pliage.
  • Rayon de croquage

Angle
