Importation de la pièce à plier

Cet onglet permet d'importer les pièces à plier dans le menu Home. L'affichage peut être filtré dans la ligne de saisie en spécifiant le nom de la pièce.

Type de fichier
  • Tous les formats de fichiers connus

    • Cette fonction permet d'importer depuis le stockage des données une pièce à plier créée au format .bpx ou .bendpart.

  • Fichiers BPX

    • Cette fonction permet d'importer depuis le stockage des données une pièce à plier créée avec MMC ByBend ou BySoft 6.x.

  • Fichiers Bendpart

    • Cette fonction permet d'importer depuis le stockage des données une pièce à plier créée avec le programme de CFAO BySoft 7.

  • Fichiers DXF

    • Cette fonction permet d'importer depuis le stockage des données une pièce à plier au format DXF.

Trier par
  • Nom

    • Trie les pièces à plier sur le nom des pièces.

  • Date

    • Trie les pièces à plier sur la date de création.

  • Dimension

    • Trie les pièces à plier sur les dimensions du fichier.

  • Croissant

    • En commençant par le début de l'alphabet.

  • Croissant

    • En commençant par la fin de l'alphabet.

Importer

Importe les pièces à plier sélectionnées dans le pool de pièces du programme. Une seule pièce peut être importée.

Conflits lors de l'importation

En cas de conflits de nommage survenant lors de l'importation automatique des fichiers *.bendpart, ceux-ci sont affichés.

Différentes solutions sont possibles :

  • Supprimer le fichier d'origine. Le fichier n'est pas importé.
  • Importer le fichier et remplacer la pièce de pliage existante, portant le même nom.

    • La pièce déjà chargée est déchargée et supprimée automatiquement. La nouvelle pièce est ensuite importée et est automatiquement chargée sous le même nom.

    Attention! Toutes les modifications de la technologie, les corrections, etc. de l'ancienne pièce sont alors perdues.

Définition des favoris

Enregistre le chemin sélectionné dans les favoris. Ce chemin est toujours affiché sur cet onglet sous forme de chemin par défaut.

Tip: Les favoris permettent de trouver plus simplement les pièces à plier à importer enregistrées dans un chemin par défaut.
