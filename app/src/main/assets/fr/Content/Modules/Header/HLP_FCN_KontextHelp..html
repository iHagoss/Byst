Aide contextuelle

Cette fonction affiche la fenêtre d'aide contextuelle sur le côté droit du programme. Les fenêtres du programme sont réduites à la proportion correspondante. Vous obtenez ainsi des informations ciblées sans recouvrir des fonctions importantes dans le programme.

L'affichage des différents thèmes de l'aide est contextuelle en fonction de la vue ou de la fonction sélectionnée dans le programme.

Info: Tapotez sur la ligne d'en-tête de la boîte de dialogue d'aide pour masquer à nouveau la boîte de dialogue.

Symbole Signification
Ouvre ou ferme un texte déroulant
Ouvre la boîte de dialogue de l'imprimante pour imprimer le thème affiché
Fait défiler en arrière le début du thème. Le bouton ne s'affiche que si le thème est plus grande que la zone d'affichage verticale dans la boîte de dialogue d'aide.
Désactiver l'aide contextuelle

L'icône Epingle dans la ligne d'en-tête de la boîte de dialogue d'aide permet de désactiver l'aide contextuelle. Lorsque l'aide contextuelle est désactivée, le contenu est maintenu y compris lorsque l'on passe à une autre boîte de dialogue ou une autre fonction.

Tip: La désactivation de l'aide contextuelle est nécessaire pour les descriptions de processus, pour lesquelles différentes vues et fonctions doivent être sélectionnées dans le programme.

Symbole Signification
Aide contextuelle activée
Aide contextuelle désactivée

 

Naviguer dans l'aide contextuelle

Différentes possibilités sont disponibles dans l'aide contextuelle pour la navigation, indépendamment du contexte actuel.

  • Breadcrums

    • Les Breadcrums (miettes de pain) montrent le chemin vers le thème actuel dans l'aide. Tapotez sur une entrée soulignée pour passer au thème associé.

  • Boutons des symboles

    • Tapotez sur un bouton d'icône entouré de bleu pour passer au thème correspondant.
