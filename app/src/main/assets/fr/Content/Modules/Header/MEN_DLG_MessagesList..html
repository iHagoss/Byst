Liste des messages

WARNING: Les opérations sur les équipements électriques ne doivent être réalisés que par un électricien dans le respect des règles électrotechniques.

Tous les messages en cours sont affichés sous forme tabulaire dans la liste des messages. Les icônes en regard du message indiquent la catégorie et le module d'origine du message.

Symbole Catégorie
Il n'y a aucun message en attente.
Il y a un message d'état en attente. Les messages d'état informent l'opérateur sur le processus en cours sur la machine.

Il y a un message d'avertissement en attente. Les avertissements font référence à des défauts ou à des erreurs système, l'installation est opérationnelle de manière limitée.
Il y a un message d'erreur en attente. Les messages d'erreur font référence à des défauts ou à des erreurs système. L'installation n'est pas opérationnelle.

Fonctions dans la liste des messages

Symbole Signification
Réinitialise les messages en cours lorsque leur cause est corrigée.
