Sistema de medición angular

En este registro se configura y ajusta el sistema de medición de ángulo. Este registro se visualiza si la máquina dobladora seleccionada está equipada con un sistema de medición de ángulo (LAMS Véase el sistema de medición de ángulo).

Tip: Utilice el sistema de medición de ángulo para el doblado de ángulos muy precisos.

Vista 3D

En la vista 3D se muestra el proceso de doblado actualmente seleccionado con el sistema de medición de ángulo activado.

En la primera tabla debajo de la ventana de visualización se muestra el estado del sistema de medición de ángulo para cada proceso de doblado. La segunda tabla muestra la posición y el modo del sistema de medición de ángulo.

Tip: Pulsando sobre la entrada de la tabla se abre el diálogo para editar el sistema de medición de ángulo.

Posicionar el sistema de medición de ángulo con el pedal

Si se desplaza el sistema de medición de ángulo a otra posición para el próximo proceso de doblado es necesario que el usuario confirme el posicionamiento del sistema de medición de ángulo pulsando sobre el pedal. Luego se desplaza el sistema de medición de ángulo a la posición correspondiente.

La confirmación sirve para evitar colisiones del sistema de medición de ángulo con la pieza a doblar.

NOTE: En la base de datos de dobladuras se determina la medida a partir de la cual es necesaria una confirmación con el pedal.

Explicación de las columnas individuales:

Estado

Muestra el estado del sistema de medición de ángulo en la secuencia de doblado. Se muestra respectivamente el estado de la dobladura actual.

Símbolo Descripción

Ángulo

Muestra el ángulo de la dobladura. El ángulo 0 sólo se utiliza para el doblado final de una dobladura plana.

Operación de doblado

Muestra el tipo de doblado en el proceso de doblado.

  • Doblado normal El doblado normal es el método usual de hacer un doblado. El doblado queda determinado por el ángulo de doblado y un ángulo de predoblado opcional.
  • Dobladura plana Este tipo de doblado pliega la chapa con un ángulo de doblado de 180° dejándola plana. El proceso de doblado se ejecuta en los dos pasos consecutivos «Predoblado» y «Dobladura plana». Estos pasos pueden ejecutarse también independientemente.
  • Predoblado Este tipo de doblado hace una dobladura en dos pasos. El doblado se dobla primero con un ángulo de predoblado y posteriormente se acaba de doblar con el ángulo final. Utilice el predoblado en piezas complejas a doblar para evitar colisiones durante el proceso de doblado.
  • Doblado auxiliar El doblado auxiliar se utiliza en determinadas formas de doblado de forma provisional, que luego se desdobla, para posibilitar dar la forma final a una pieza a doblar.
  • Doblado El doblado es un elemento de la programación de perfiles y designa un cambio de forma en la pieza a mecanizar. Es posible asignar varios atributos a un doblado. Longitud de lado, ángulo de doblado, radio de doblado (optativo), ángulo de predoblado (optativo), atributos adicionales (optativo). Modo

Indica si el sistema de medición de ángulo está conectado o desconectado. Estado desconectado sólo puede ajustarse la posición del sistema de medición de ángulo.

Estacionado

Indica si el sistema de medición de ángulo está en la posición de estacionamiento o posicionado. Si el sistema de medición de ángulo está en posición de estacionamiento se desactivan todas las demás opciones.

Posición LZB

Muestra la distancia del punto de referencia del sistema de medición de ángulo posterior respecto al borde exterior izquierdo de la mesa de prensa.

Posición LZF

Muestra la distancia del punto de referencia del sistema de medición de ángulo delantero respecto al borde exterior izquierdo de la mesa de prensa.
