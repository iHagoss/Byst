Funciones

Con las funciones de la barra de funciones se asigna la herramienta de doblado correspondiente a los procesos de doblado. La asignación puede hacerse manual o automáticamente. Las herramientas de doblado ya puestas pueden editarse posteriormente.

Puede disponer de las siguientes funciones:

Funciones Significado
Tecnología Con esta función se determina toda la tecnología de doblado y se le asigna automáticamente a la pieza a doblar.
Automático Con esta función se asignan las herramientas estándar seleccionadas a cada proceso de doblado según los criterios de la caja de diálogo Ajustes de autómata.
Completar Con esta función se calculan todas las herramientas que no estén puestas todavía. Las herramientas ya puestas se conservan y no tienen que volver a calcularse.
Comprobar Con esta función se comprueba si los procesos pueden doblarse con las herramientas asignadas. El resultado de la comprobación se muestra en una ventana de mensajes.

Info: Las herramientas que ya tengan el estado de OK () no se comprueban más.
Ajustes Se abre un diálogo en el que pueden modificarse los ajustes para la herramienta estándar.
Editar Se abre un diálogo en el que puede modificarse el ajuste de herramienta para el proceso de doblado actual.
