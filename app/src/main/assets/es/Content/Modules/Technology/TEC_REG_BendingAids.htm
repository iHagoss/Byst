Dispositivos auxiliares de doblar

En este registro se configuran y ajustan todos los dispositivos auxiliares para doblar disponibles. Los dispositivos auxiliares para doblar se utilizan como apoyo durante el proceso de doblado de grandes chapas voluminosas o pesadas.

Vista 3D

En la vista 3D se muestra el proceso de doblado actualmente seleccionado con los dispositivos auxiliares para doblar activados. Debajo de la ventana de visualización se muestra en forma tabular el estado de los dispositivos auxiliares para doblar de cada proceso de doblado.

Tip: Pulsando sobre el correspondiente dispositivo auxiliar para doblar de la tabla se abre el diálogo para editar los dispositivos auxiliares para doblar.

Ángulo

Muestra el ángulo de la dobladura. El ángulo 0 sólo se utiliza para el doblado final de una dobladura plana.

Operación de doblado

Muestra el tipo de doblado en el proceso de doblado.

  • Doblado normal El doblado normal es el método usual de hacer un doblado. El doblado queda determinado por el ángulo de doblado y un ángulo de predoblado opcional.
  • Dobladura plana Este tipo de doblado pliega la chapa con un ángulo de doblado de 180° dejándola plana. El proceso de doblado se ejecuta en los dos pasos consecutivos «Predoblado» y «Dobladura plana». Estos pasos pueden ejecutarse también independientemente.
  • Predoblado Este tipo de doblado hace una dobladura en dos pasos. El doblado se dobla primero con un ángulo de predoblado y posteriormente se acaba de doblar con el ángulo final. Utilice el predoblado en piezas complejas a doblar para evitar colisiones durante el proceso de doblado.
  • Doblado auxiliar El doblado auxiliar se utiliza en determinadas formas de doblado de forma provisional, que luego se desdobla, para posibilitar dar la forma final a una pieza a doblar.
  • Doblado El doblado es un elemento de la programación de perfiles y designa un cambio de forma en la pieza a mecanizar. Es posible asignar varios atributos a un doblado. Longitud de lado, ángulo de doblado, radio de doblado (optativo), ángulo de predoblado (optativo), atributos adicionales (optativo). Sincrónico hasta ángulo

Muestra el ángulo hasta el que los dispositivos auxiliares para doblar van sincrónicamente con el movimiento del portapunzón y el retroceso con velocidad reducida después del proceso de doblado (See «Editar»).

El ángulo se muestra también si el dispositivo auxiliar para doblar está en posición de estacionamiento.

Dispositivo auxiliar para doblar delante / atrás

Muestra los dispositivos auxiliares para doblar disponibles. Pulsando sobre un dispositivo auxiliar para doblar cambia a color azul en la ventana de visualización.

  • 1 = Dispositivo auxiliar para doblar izquierdo
  • 2 = Dispositivo auxiliar para doblar derecho

Estacionado

Estando activado el atributo significa que el dispositivo auxiliar de doblado está en posición de estacionamiento

Activado

Estando activado el atributo significa que el dispositivo auxiliar de doblado se está usando en ese proceso.

Posición H

Muestra la altura del dispositivo auxiliar de doblado en el correspondiente proceso de doblado.

Posición W

Muestra la distancia del punto de referencia del correspondiente dispositivo auxiliar para doblar respecto al borde exterior izquierdo de la mesa de prensa.

Dicha posición se muestra también si el dispositivo auxiliar de doblado está en posición pero no activado.
