Topes

En este registro se generan los topes para todos los procesos de doblado. Los topes se necesitan para la colocación precisa de la pieza a doblar antes de ejecutar el doblado. Los topes pueden hacerse utilizando como mínimo un Talón de tope a la derecha y otro a la izquierda.

Debajo de la ventana de visualización se muestran las entradas definidas y el estado de los topes para cada proceso de doblado.

Tip: Pulsando sobre el correspondiente tope de la tabla se abre el diálogo para editar los topes.

Posicionar los topes con el pedal

Si se desplazan los topes a otra posición hacia delante en dirección al usuario para el próximo proceso de doblado, entonces es necesario que el usuario confirme el posicionamiento de los topes pulsando sobre el pedal. Luego se desplazan los topes a la posición correspondiente.

La confirmación sirve para la seguridad del usuario para evitar que el dispositivo auxiliar para doblar choque con el usuario al posicionar la pieza a doblar si se usan lados largos en el área posterior de la máquina.

NOTE: En la base de datos de dobladuras se determina la medida a partir de la cual es necesaria una confirmación con el pedal.

Ángulo

Operación de doblado

Muestra el tipo de doblado en el proceso de doblado.

  • Doblado normal El doblado normal es el método usual de hacer un doblado. El doblado queda determinado por el ángulo de doblado y un ángulo de predoblado opcional.
  • Dobladura plana Este tipo de doblado pliega la chapa con un ángulo de doblado de 180° dejándola plana. El proceso de doblado se ejecuta en los dos pasos consecutivos «Predoblado» y «Dobladura plana». Estos pasos pueden ejecutarse también independientemente.
  • Predoblado Este tipo de doblado hace una dobladura en dos pasos. El doblado se dobla primero con un ángulo de predoblado y posteriormente se acaba de doblar con el ángulo final. Utilice el predoblado en piezas complejas a doblar para evitar colisiones durante el proceso de doblado.
  • Doblado auxiliar El doblado auxiliar se utiliza en determinadas formas de doblado de forma provisional, que luego se desdobla, para posibilitar dar la forma final a una pieza a doblar.
  • Doblado El doblado es un elemento de la programación de perfiles y designa un cambio de forma en la pieza a mecanizar. Es posible asignar varios atributos a un doblado. Longitud de lado, ángulo de doblado, radio de doblado (optativo), ángulo de predoblado (optativo), atributos adicionales (optativo). Engatillar

Muestra si las esquinas de la pieza a doblar están engatilladas. Se denomina engatillar a la sujeción de dos puntos en los talones de tope para topes angulares (See «Engatillar»).

Símbolo Descripción
El engatillado está activado. El tope engatilla correctamente en una esquina de la pieza.
El engatillado está desactivado. El tope toca un borde recto de la pieza.
El engatillado está desactivado. El tope toca un borde oblicuo de la pieza.
El símbolo caracteriza al tope como tope maestro. El tope maestro determina la posición del eje R para todos los topes existentes.

NOTE: El tope maestro se utiliza sólo con topes que dispongan de una eje R común.
