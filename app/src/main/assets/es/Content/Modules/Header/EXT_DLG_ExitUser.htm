Fin de sesión de usuario

Esta función cambia al menú Cierre de sesión. Seleccionar el botón Fin de sesión para finalizar la sesión del usuario.

Forma de proceder:

  1. Seleccionar botón.
  2. Seleccionar botón. Se muestra el diálogo de inicio de sesión.
  3. Seleccionar el botón Fin de sesión en el diálogo de inicio de sesión. Se cierra la sesión del usuario actual.
Cambiar la versión de idioma del usuario que ha iniciado sesión
  1. Seleccionar botón.
  2. Seleccionar botón.
  3. Seleccionar botón.
  4. Pulsar el botón Seleccionar idioma.
  5. Seleccionar el idioma deseado en el diálogo de selección. El programa se adapta automáticamente al idioma elegido.
