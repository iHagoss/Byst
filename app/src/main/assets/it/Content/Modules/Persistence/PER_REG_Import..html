Importare pezzo da piegare

In questa scheda, è possibile importare le parti di piegatura nel menu Home. Nella riga di input, è possibile filtrare il display specificando il nome del pezzo.

Tipo di File
  • Tutti i formati di file conosciuti

    • Con questa funzione è possibile importare da un archivio dati un pezzo da piegare creato in formato .bpx o .bendpart.

  • File BPX

    • Con questa funzione è possibile importare da un archivio dati un pezzo da piegare creato con MMC ByBend o BySoft 6.x.

  • File Bendpart

    • Con questa funzione è possibile importare da un archivio dati un pezzo da piegare creato con il programma CAD/CAM BySoft 7.

  • File DXF

    • Con questa funzione, è possibile importare da un archivio dati un pezzo da piegare in formato DXF con le relative informazioni per la piegatura.

Ordinare per
  • Nome

    • Ordina i pezzi da piegare in base al nome.

  • Data

    • Ordina i pezzi da piegare in base alla data di creazione.

  • Dimensione

    • Ordina i pezzi da piegare in base alle dimensioni del file.

  • ascendente

    • In ordine alfabetico ascendente

  • ascendente

    • In ordine alfabetico discendente

Importare

Importa i pezzi da piegare selezionati nel pool del programma. È possibile importare solo un pezzo alla volta.

Conflitti durante l'importazione

Con l’importazione automatica di file *.bendpart verrebbero visualizzati eventuali conflitti di nome.

Esistono le seguenti soluzioni:

  • Eliminare il file sorgente. Il file non viene importato.
  • Importare il file e sovrascrivere il pezzo da piegare esistente con lo stesso nome.

    • Il pezzo già caricato viene automaticamente scaricato ed eliminato. Quindi, il nuovo pezzo viene importato e caricato automaticamente con lo stesso nome.

    Attention! Tutte le modifiche alla tecnologia, correzioni, ecc. del pezzo caricato andranno perse.

Imposta come preferito

Salva il percorso selezionato come preferito. Questo percorso viene sempre visualizzato come predefinito in questa scheda.

Tip: I preferiti facilitano la ricerca di pezzi da piegare per l'importazione archiviati in un percorso predefinito.
