Production - Automatique

La production des pièces de pliage est réalisée dans ce module. Les données lues automatiquement par la pièce de pliage sélectionnée sont appliquées et les pièces de pliage sont traitées en conséquence. Les données chargées peuvent être modifiées si besoin.

La fenêtre indique dans la vue isométrique la première étape du processus de la pièce de pliage sélectionnée. Si la machine est correctement installée et que la pièce peut être produite, la fenêtre indique sur les bords inférieur et supérieur une barre verte. L'icône d'actionnement de la pédale est affichée dans le coin inférieur droit.

Informations dans la zone texte de la fenêtre d'affichage :

  • Nom du produit
  • Quantité
  • Compteur de reste (activé si la quantité de pièces de consigne est définie)
  • Temps restant (activé dès que la première pièce a été produite)

Cette zone affiche l'étape de processus, l'angle de pliage ainsi que la quantité. Lorsque le nombre de pièces de consigne à a été programmé, le temps restant pour l'usinage est également affiché.

Mode de travail mode automatique et pas à pas

Bouton de sélection entre le mode automatique et le mode pas à pas.

Lors du traitement des processus, il est possible d'alterner entre le mode Automatique et Pas à pas en tapotant successivement sur ce bouton.

Info: Lors du démarrage de la plieuse, celle-ci travaille toujours en mode automatique.

  • Mode automatique

    • Si le bouton central n'est pas marqué, le mode de travail est en mode automatique. L'entrée Production en mode automatique apparaît dans la zone de texte de la ligne d'en-tête.

    Le programme démarre avec le processus actuel et traite tous les processus actuels. L'opération se répète jusqu'à ce que la quantité de consigne programmée soit atteinte. Si aucune quantité n'a été programmée, le programme est répétée jusqu'à ce que vous interrompiez l'opération.

  • Pas à pas

    • Si le bouton central est marqué en vert, le mode de travail est en mode pas à pas. L'entrée Production en mode pas à pas apparaît dans la zone de texte de la ligne d'en-tête.

    On utilise toujours le même processus de pliage jusqu'à ce que le prochain processus de pliage soit sélectionné. Il est possible de naviguer dans les processus et de les traiter dans l'ordre.

    Le compteur de pièces n'est pas activé dans ce mode, même si une quantité est programmée. Si vous actionnez à nouveau le bouton, le logiciel repasse en mode automatique.
