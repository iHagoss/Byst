Fonctions

Les fonctions dans la barre de fonctions permettent d'affecter l'outil de pliage correspondant aux processus de pliage. L'affectation peut être réalisée de manière manuelle ou automatique. Les outils de pliage définis peuvent être modifiés ultérieurement.

Différentes fonctions sont disponibles :

Fonctions Signification
Corriger Cette boîte de dialogue permet de modifier les valeurs d'axe et les paramètres de processus pendant la production.
Compensation Cette boîte de dialogue permet d'afficher les valeurs de compensation de différents paramètres.
Tableau de pliage  
Pliage de finition Lors d'une correction de l'angle dans le sens positif, le fonction permet d'exécuter un pliage de finition.
