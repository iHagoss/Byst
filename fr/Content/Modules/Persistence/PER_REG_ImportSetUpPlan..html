Importer

Cette boîte de dialogue permet d'importer le plan de positionnement d'une pièce à plier présente vers la pièce à plier actuelle.

Toutes les pièces à plier présente dans le pool de pièces sont affichées sous forme d'images 2D dans la vue principale. L'affichage peut être filtré dans la ligne de saisie en spécifiant le nom de la pièce.

Info: Le filtre est utilisé pendant la saisie.

Procédure :

  1. Sélectionner la pièce à plier dont le plan de positionnement doit être importé.
  2. Sélectionner le bouton Importer. Le plan de positionnement est importé et est affecté à la pièce à plier en tant que nouveau plan de positionnement.

Info: Les plans de positionnement présents sont conservés et doivent être supprimés manuellement.

Trier par

Les pièces à plier listées peuvent être triées sur les noms ci-après.

  • Nom

    • Affiche le nom de la pièce.

  • Description

    • Ce champ permet de spécifier une description plus précise de l'objet actuel.

  • Date

    • Affiche la date de la dernière actualisation.

  • Info 1-3

    • Ce champ peut comporter des informations sur l'objet en cours.

  • Croissant

    • En commençant par le début de l'alphabet.

  • Décroissant

    • En commençant par la fin de l'alphabet.

Importer

Importe le plan de positionnement de la pièce à plier sélectionnée dans la pièce à plier actuelle.

Filtre de recherche

Cette boîte de dialogue permet de réaliser des réglages de filtre pour la boîte de dialogue de saisie.

  • Rechercher dans

    • Toutes les options sélectionnées sont prises en compte pour la saisie du texte dans la boîte de dialogue de saisie.

  • Opérateur de recherche

    • Définit le lien entre les options du filtre.

  • Créé entre

    • Prend en compte toutes les pièces à plier créées entre les dates indiquées.

  • Modifié entre

    • Prend en compte toutes les pièces à plier modifiées entre les dates indiquées.

Précédent

Referme la boîte de dialogue et revient à la vue Plan de positionnement. Aucun plan de positionnement n'est importé.
