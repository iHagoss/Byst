Usinage

Ce dialogue permet de modifier la position du système de mesure des angles ou de le redéfinir pour le processus de pliage sélectionné. Les indications invalides sont signalées par un symbole d'erreur .

Fonction

Cette rubrique permet de paramétrer les fonctions du système de mesure des angles.

Mode

Cette option permet d'activer ou de désactiver le système de mesure des angles selon des critères donnés.

  • Activé

    • Active le système de mesure angulaire pour tous les processus de pliage.

  • Premier activé

    • Active la mesure des angles pour le premier processus de pliage des angles de pliage identiques.

  • Désactivé

    • Désactive le système de mesure des angles. Il est uniquement possible de configurer les axes du système de mesure des angles.

  • Récupérer

    • Valide les valeurs mesurées du processus de pliage indiquées dans le champ Validation du processus.

Validation du processus

Cette option permet de valider les valeurs mesurées par le processus de pliage sélectionné.

Capteur

Les capteurs sélectionnés sont utilisés pour la mesure des angles. Les deux capteurs doivent être utilisés par défaut afin d'améliorer la précision.

Chaque énième pièce de pliage

Dans une série de pièces de pliage identiques, il est possible d'indiquer ici après combien de pièces produites une pièce supplémentaire doit être contrôlée par le système de mesure angulaire.

Axes

Cette rubrique permet de positionner le système de mesure des angles le long du logement de l'outil.

Utilisation

Cette option permet de positionner le système de mesure des angles ou de le déplacer vers la position de garage. Si l'option Immobilisé est activée, toutes les autres options du système de mesure des angles sont désactivées.

Position LZB/F

Cette option permet de positionner le système de mesure des angles sur la station de pliage du processus de pliage sélectionné.

  • B = Arrière
  • F = Avant
Déplacement

Cette option permet de déplacer le système de mesure des angles en tapotant et en tirant à la main à la position souhaitée dans la vue 3D (See «Utiliser l'écran tactile»).

Compensation de l'effet de retour

Cette rubrique permet de configurer la correction de l'effet de retour du matériel pendant le pliage. Elle n'est activée que le mode prend la valeur Activé sous Fonction.

Mode

La mesure de l'effet de retour est requise pour compenser le retour du matériel pendant le pliage.

  • Mesure activée

    • Ceci permet de définir automatiquement l'angle de retour de pliage.

  • Base de données

    • L'angle de retour de pliage est utilisé à partir de la base de données.

  • Manuel

    • L'angle de retour de pliage doit être indiqué manuellement à l'aide de l'option Angle.

Angle attendu

Définit l'angle attendu pour la mesure de compensation de l'effet de retour élastique. Si l'angle n'est plus dans la plage indiquée, un message d'erreur est émis.

  • Base de données

    • L'angle attendu est utilisé à partir de la base de données.

  • Manuel

    • L'angle attendu peut être entré dans le champ situé au-dessous.

Angle

Si vous sélectionnez l'attribut Manuel pour l'optionAngle attendu, vous devez entrer dans ce champ l'angle de l'effet de retour élastique.
