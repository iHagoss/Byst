Modifier le segment

Cette boîte de dialogue permet d'ajouter tous les segments d'outils disponibles d'une station.

  • Segment Un segment est un outil de pliage utilisé comme station de pliage ou partie d'une station de pliage.

    • Cette option permet d'ajouter le segment sélectionné de la station de pliage.

  • Segment vide Un segment vide est une rubrique définie librement dans une station de pliage. Les segments vides sont utilisés pour les lignes de pliage reliées entre deux côtés lorsqu'il y a un contour qui ne doit pas être plié. Un segment vide doit toujours être entouré par deux segments normaux.

    • Cette option permet d'ajouter un segment vide de longueur quelconque à la station.
