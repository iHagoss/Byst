Plan positionnement outils

Cette vue représente graphiquement les informations sur les stations paramétrées.

Les différentes stations sont représentées dans la partie inférieure de l'écran. Lorsque vous touchez les outils supérieurs ou inférieurs d'une vue de station, ceux-ci apparaissent en surbrillance verte. Les sections des outils sélectionnés sont représentées graphiquement et les informations correspondantes sont affichées.

Les informations affichées dans les lignes d'information Info 1-3 sont entrées préalablement dans l'Administrateur, à la rubrique correspondante et pour l'outil sélectionné.

  • Le bouton Garer ouvre la boîte de dialogue pour le déplacement des composants à la position de garage (See «Garage»).
  • Le bouton Laser à ligne de pliage ouvre la boîte de dialogue pour le réglage du faisceau laser (See «Laser à ligne de pliage»).
