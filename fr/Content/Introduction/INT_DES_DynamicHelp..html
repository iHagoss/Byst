Afficher l'aide en ligne

L'aide en ligne fait partie intégrante du programme. En fonction de la langue du système d'exploitation, le programme et l'aide en ligne sont affichés dans la langue définie.

Possibilités d'affichage de l'aide

Ce programme permet d'appeler l'aide à partir du menu principal de deux manières différentes.

Symbole Signification
Aide générale

Affiche l'aide dans un système d'aide spécifique. Différentes possibilités sont disponibles pour la navigation dans le système d'aide :

  • Sommaire
  • Index
  • Glossaire
  • Recherche en texte intégral
Aide contextuelle

Affiche l'aide dans une boîte de dialogue d'aide intégrée au programme. Le contenu est affiché en fonction du contexte de l'élément logiciel sélectionné.

Sélectionnez l'épingle dans la boîte de dialogue d'aide pour activer ou désactiver la fonction contextuelle.

  • Aide contextuelle activée
  • Aide contextuelle désactivée
Appeler l'aide générale
  1. Sélectionner le bouton.
  2. Sélectionner le bouton. L'aide s'ouvre dans une application spécifique.
Appeler l'aide contextuelle
  1. Sélectionner le bouton.
  2. Sélectionner le bouton. Les informations sur la vue ou les fonctions sélectionnées sont affichées dans la boîte de dialogue qui s'affiche.
Afficher l'aide sur les fonctions dans la ligne d'en-tête
  1. Sélectionner le bouton.
  2. Sélectionner le bouton. La boîte de dialogue de l'aide est affichée.
  3. Tapoter sur l'icône souhaitée du menu dans la ligne d'en-tête. Le thème de l'aide associé à la fonction est affiché.
  4. Faire glisser le doigt d'aide sans l'éloigner de l'écran sur une zone de la boîte de dialogue qui n'est pas recouverte par le menu (See «Glisser»).
  5. Retirer le doigt de l'écran.
  6. Tapoter sur l'icône correspondant à la fonction souhaitée dans l'aide.
